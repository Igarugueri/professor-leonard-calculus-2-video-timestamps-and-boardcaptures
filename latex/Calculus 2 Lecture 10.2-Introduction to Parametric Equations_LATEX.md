в—Ҹ [0:00](https://www.youtube.com/watch?v=d4KADBFqpR0&t=0). Introduction to a вҖңBrave New WorldвҖқ of Calculus.
     в—ү Goal:

пј‘пјҺпј’  пј°пҪ’пҪҸпҪӮпҪҢпҪ…пҪҚпҪ“гҖҖпҪ—пҪүпҪ”пҪҲгҖҖпјӨпҪ…пҪҶпҪүпҪҺпҪүпҪҺпҪҮгҖҖпј°пҪҢпҪҒпҪҺпҪ…гҖҖпјЈпҪ•пҪ’пҪ–пҪ…пҪ“гҖҖпҪҒпҪ“гҖҖпјҜпҪ’пҪ„пҪүпҪҺпҪҒпҪ’пҪҷгҖҖпјҰпҪ•пҪҺпҪғпҪ”пҪүпҪҸпҪҺпҪ“

в—Ҹ [2:20](https://www.youtube.com/watch?v=d4KADBFqpR0&t=140). Problems with defining a plane curve like a normal function.
     в—ү Problem 1:
          в—Ӣ many plane curves are not functions
          в—Ӣ for example, circles fail the vertical line test
     в—ү Problem 2:
          в—Ӣ many curves cannot be explicitly defined in terms of a single variable
     в—ү Problem 3:
          в—Ӣ an ordinary rectangular equation does not tell where the object is at a given time
     в—ү Problem 4:
          в—Ӣ an ordinary function does not give the direction of motion
     в—ү Main conclusion:
          в—Ӣ rectangular equations often describe the path
          в—Ӣ but they do not describe the motion along the path

в—Ҹ [9:42](https://www.youtube.com/watch?v=d4KADBFqpR0&t=582). Definition вҖ” Parametric equations.
     в—ү Main idea:
          в—Ӣ define both $рқ’ҷ$ and $рқ’ҡ$ as functions of a third variable
     в—ү Standard form:
          в—Ӣ $рқ’ҷ=рқ’Ү(рқ‘»)$
          в—Ӣ $рқ’ҡ=рқ’Ҳ(рқ‘»)$
     в—ү Meaning:
          в—Ӣ for each value of $рқ‘»$, the equations produce one point $(рқ’ҷ,рқ’ҡ)$
     в—ү Important:
          в—Ӣ $рқ’ҷ$ and $рқ’ҡ$ are not defined in terms of each other
          в—Ӣ both are defined in terms of the same parameter
     в—ү For evample at $рқ‘»=1$ tour x is 3 and youy y is 5


в—Ҹ [14:50](https://www.youtube.com/watch?v=d4KADBFqpR0&t=890). Core terminology of parametric systems.
     в—ү Parameter:
          в—Ӣ $рқ‘»$
     в—ү Meaning of the parameter:
          в—Ӣ the independent variable on which both $рқ’ҷ$ and $рқ’ҡ$ depend
     в—ү Parameter interval:
          в—Ӣ $\mathcal{I}=[рқ’Ӯ,рқ’ғ]$
     в—ү Meaning of the interval:
          в—Ӣ it tells where the motion starts and where it stops
     в—ү Orientation:
          в—Ӣ the direction in which the object moves as the parameter increases


в—Ҹ [15:50](https://www.youtube.com/watch?v=d4KADBFqpR0&t=950). Interpretation of the parameter.
     в—ү $рқ‘»$ is usually thought of as time
     в—ү This is only an interpretation:
          в—Ӣ $рқ‘»$ may be negative
          в—Ӣ it is still just the parameter


в—Ҹ [17:03](https://www.youtube.com/watch?v=d4KADBFqpR0&t=1023). Initial and terminal points.
     в—ү $рқ‘»$ ГЁrtenciente a $[рқ’Ӯ,рқ’ғ]$ tiene un nitial point a y terinal oint b
     в—ү Initial point:
          в—Ӣ iМІfМІ $рқ‘»=рқ’Ӯ$
               в–  tМІhМІeМІnМІ $(рқ’ҷ,рқ’ҡ)=(рқ’Ү(рқ’Ӯ),рқ’Ҳ(рқ’Ӯ))$
     в—ү Terminal point:
          в—Ӣ iМІfМІ $рқ‘»=рқ’ғ$
               в–  tМІhМІeМІnМІ $(рқ’ҷ,рқ’ҡ)=(рқ’Ү(рқ’ғ),рқ’Ҳ(рқ’ғ))$
     в—ү Important:
          в—Ӣ the curve is traced from the initial point to the terminal point
          в—Ӣ si empezamos en un punto y terminamos en otro esto da  this gives the trajectory a specific orientation

в—Ҹ [23:11](https://www.youtube.com/watch?v=d4KADBFqpR0&t=1391). рҹ§© Example 1 вҖ” $рқ’ҷ=рқ‘»^{2}-4$, $рқ’ҡ=2рқ’•$ on $-1\leq рқ‘»\leq 2$.
     в—ү вқ¶ Make a $рқ’•$-table.
          в—Ӣ choose values of $рқ‘»$ in the interval
          в—Ӣ compute the corresponding points $(рқ’ҷ,рқ’ҡ)$
          в—Ӣ for example:
               $\rule{0pt}{}$
               в–  $рқ‘»=-1$    вҶ’   $(рқ’ҷ,рқ’ҡ)=(-3,-2)$
               $\rule{0pt}{}$
               в–  $рқ‘»=-\dfrac{1}{2}$  вҶ’   $\left(рқ’ҷ,рқ’ҡ\right)=\left(-\dfrac{15}{4},-1\right)$
               $\rule{0pt}{}$
               в–  $рқ‘»=0$     вҶ’   $(рқ’ҷ,рқ’ҡ)=(-4,0)$
               $\rule{0pt}{}$
               в–  $рқ‘»=\dfrac{1}{2}$   вҶ’   $\left(рқ’ҷ,рқ’ҡ\right)=\left(-\dfrac{15}{4},1\right)$
               $\rule{0pt}{}$
               в–  $рқ‘»=1$     вҶ’   $(рқ’ҷ,рқ’ҡ)=(-3,2)$
               $\rule{0pt}{}$
               в–  $рқ‘»=2$     вҶ’   $(рқ’ҷ,рқ’ҡ)=(0,4)$
               $\rule{0pt}{}$
     в—ү вқ· Plot the points in the $рқ’ҷрқ’ҡ$-plane.
     в—ү вқё Connect the points with a smooth curve.
     в—ү вқ№ Identify the motion.
          в—Ӣ Initial Point:
               $\rule{0pt}{}$
               в–  $рқ‘»=-1$  вҶ’  $(рқ’ҷ,рқ’ҡ)=(-3,-2)$
               $\rule{0pt}{}$
          в—Ӣ Terminal Point:
               $\rule{0pt}{}$
               в–  $рқ‘»=2$  вҶ’  $(рқ’ҷ,рқ’ҡ)=(0,4)$
               $\rule{0pt}{}$
          в—Ӣ Orientation:
               в–  the motion goes upward along the curve


в—Ҹ [33:04](https://www.youtube.com/watch?v=d4KADBFqpR0&t=1984). Eliminating the parameter.
     в—ү Solve for $рқ‘»$ from $рқ’ҡ=2рқ’•$:
     $\rule{0pt}{}$
          в—Ӣ $рқ‘»=\dfrac{рқ’ҡ}{2}$
          $\rule{0pt}{}$
     в—ү Substitute into $рқ’ҷ=рқ’•^{2}-4$:
     $\rule{0pt}{}$
          в—Ӣ $рқ’ҷ=\left(\dfrac{рқ’ҡ}{2}\right)^{2}-4$
          $\rule{0pt}{}$
     в—ү Rectangular form:
     $\rule{0pt}{}$
          в—Ӣ $рқ’ҷ=\dfrac{рқ’ҡ^{2}}{4}-4$
          $\rule{0pt}{}$
     в—ү Warning:
          в—Ӣ this rectangular equation describes the whole parabola
          в—Ӣ but the parameter interval gives only a specific piece of it

          

пј’пјҺпј’ - пјҘпҪҳпҪҒпҪҚпҪҗпҪҢпҪ…гҖҖпј’гҖҖвҖ”гҖҖпјіпҪ‘пҪ•пҪҒпҪ’пҪ…гҖҖпјІпҪҸпҪҸпҪ”гҖҖпјҶгҖҖпј®пҪҒпҪ”пҪ•пҪ’пҪҒпҪҢгҖҖпјӨпҪҸпҪҚпҪҒпҪүпҪҺ

в—Ҹ [рҹ“·image](../img/Calculus 2 Lecture 10.2/[37-02]-01.png)


в—Ҹ Natural domain.
     в—ү iМІfМІ no interval is given,
          в—Ӣ tМІhМІeМІnМІ use the natural domain for $рқ’•$


в—Ҹ [37:02](https://www.youtube.com/watch?v=d4KADBFqpR0&t=2222). рҹ§© Example 2 вҖ” $рқ’ҷ=\sqrt{рқ’•}$, $рқ’ҡ=рқ‘»$
     в—ү вқ¶ Find the natural domain for $рқ’•$.
          в—Ӣ since $\sqrt{рқ’•}$ is defined only for $рқ‘»\geq 0$,
               в–  the natural domain is $рқ‘»\geq 0$
               в–  so the parameter interval is $\mathcal{I}=[0,\infty)$
     в—ү вқ· Eliminate the parameter.
          в—Ӣ $рқ’ҷ=\sqrt{рқ’•}$
          в—Ӣ square both sides:
               в–  $рқ‘»=рқ’ҷ^{2}$
          в—Ӣ substitute into $рқ’ҡ=рқ’•$
               в–  $рқ’ҡ=рқ’ҷ^{2}$
     в—ү вқё Restrict the rectangular curve.
          в—Ӣ since $рқ‘»\geq 0$ and $рқ’ҷ=\sqrt{рқ’•}$,
               в–  $рқ’ҷ\geq 0$
          в—Ӣ so we keep only the right half of the parabola $рқ’ҡ=рқ’ҷ^{2}$
     в—ү вқ№ Identify the motion.
          в—Ӣ Initial Point:
               в–  $рқ‘»=0$  вҶ’  $(рқ’ҷ,рқ’ҡ)=(0,0)$
          в—Ӣ Terminal Point:
               в–  none
          в—Ӣ Orientation:
               в–  the object moves from the origin toward positive infinity along the right half of the parabola

в—Ҹ [42:10](https://www.youtube.com/watch?v=d4KADBFqpR0&t=2530). рҹ§© Example 3 вҖ” $рқ’ҷ=рқ‘»$, $рқ’ҡ=рқ‘»^{2}$.
     в—ү вқ¶ Eliminate the parameter.
          в—Ӣ since $рқ’ҷ=рқ‘»$,
               в–  substitute into $рқ’ҡ=рқ‘»^{2}$
               в–  $рқ’ҡ=рқ’ҷ^{2}$
     в—ү вқ· Identify the natural domain.
          в—Ӣ there are no restrictions on $рқ‘»$
          в—Ӣ so the parameter interval is $\mathcal{I}=(-\infty,\infty)$
     в—ү вқё Interpret the motion.
          в—Ӣ iМІfМІ $рқ‘»\to -\infty$,
               в–  tМІhМІeМІnМІ $рқ’ҷ\to -\infty$ and $рқ’ҡ\to \infty$
               $\rule{0pt}{}$
          в—Ӣ iМІfМІ $рқ‘»=0$,
               в–  tМІhМІeМІnМІ $(рқ’ҷ,рқ’ҡ)=(0,0)$
               $\rule{0pt}{}$
          в—Ӣ iМІfМІ $рқ‘»\to \infty$,
               в–  tМІhМІeМІnМІ $рқ’ҷ\to \infty$ and $рқ’ҡ\to \infty$
     в—ү Final interpretation:
          в—Ӣ the rectangular path is the same as in Example 2:
               в–  $рқ’ҡ=рқ’ҷ^{2}$
          в—Ӣ but the motion is different
          в—Ӣ the object starts high in the second quadrant
          в—Ӣ moves down toward the origin
          в—Ӣ then continues up into the first quadrant
     в—ү Main lesson:
          в—Ӣ the same rectangular curve can correspond to different motions

в—Ҹ [46:14](https://www.youtube.com/watch?v=d4KADBFqpR0&t=2774). рҹ§© Example 4 вҖ” $рқ’ҷ=рқ’Ӯ\cos(\theta)$, $рқ’ҡ=рқ’Ӯ\sin(\theta)$, with $рқ’Ӯ>0$.
     в—ү Parameter:
          в—Ӣ $\theta$ is the parameter
     в—ү Given:
          в—Ӣ $рқ’ҷ=рқ’Ӯ\cos(\theta)$
          в—Ӣ $рқ’ҡ=рқ’Ӯ\sin(\theta)$
     в—ү Isolate the trig parts:
          $\rule{0pt}{}$
          в—Ӣ $\cos(\theta)=\dfrac{рқ’ҷ}{рқ’Ӯ}$
          $\rule{0pt}{}$
          в—Ӣ $\sin(\theta)=\dfrac{рқ’ҡ}{рқ’Ӯ}$
          $\rule{0pt}{}$
     в—ү Use the Pythagorean identity:
          $\rule{0pt}{}$
          в—Ӣ $\cos^{2}(\theta)+\sin^{2}(\theta)=1$
          $\rule{0pt}{}$
     в—ү Substitute:
          $\rule{0pt}{}$
          в—Ӣ $\left(\dfrac{рқ’ҷ}{рқ’Ӯ}\right)^{2}+\left(\dfrac{рқ’ҡ}{рқ’Ӯ}\right)^{2}=1$
          $\rule{0pt}{}$
     в—ү Rectangular form:
          $\rule{0pt}{}$
          в—Ӣ $рқ’ҷ^{2}+рқ’ҡ^{2}=рқ’Ӯ^{2}$
          $\rule{0pt}{}$
     в—ү Conclusion:
          в—Ӣ the curve is a circle of radius $рқ’Ӯ$ centered at the origin


в—Ҹ [54:58](https://www.youtube.com/watch?v=d4KADBFqpR0&t=3298). Points and orientation for $\mathcal{I}=[0,\pi]$.
     в—ү Initial Point:
          $\rule{0pt}{}$
          в—Ӣ $\theta=0$  вҶ’  $(рқ’ҷ,рқ’ҡ)=(рқ’Ӯ,0)$
          $\rule{0pt}{}$
     в—ү Midpoint check:
          $\rule{0pt}{}$
          в—Ӣ $\theta=\dfrac{\pi}{2}$  вҶ’  $(рқ’ҷ,рқ’ҡ)=(0,рқ’Ӯ)$
          $\rule{0pt}{}$
     в—ү Terminal Point:
          $\rule{0pt}{}$
          в—Ӣ $\theta=\pi$  вҶ’  $(рқ’ҷ,рқ’ҡ)=(-рқ’Ӯ,0)$
          $\rule{0pt}{}$
     в—ү Orientation:
          в—Ӣ counterclockwise
     в—ү Final interpretation:
          в—Ӣ only the top half of the circle is traced

в—Ҹ [1:05:17](https://www.youtube.com/watch?v=d4KADBFqpR0&t=3917). рҹ§© Example 5 вҖ” $рқ’ҷ=4\cos(\theta)$, $рқ’ҡ=3\sin(\theta)$, with $\mathcal{I}=[0,2\pi]$.
     в—ү Given:
          в—Ӣ $рқ’ҷ=4\cos(\theta)$
          в—Ӣ $рқ’ҡ=3\sin(\theta)$
     в—ү Isolate the trig parts:
          $\rule{0pt}{}$
          в—Ӣ $\cos(\theta)=\dfrac{рқ’ҷ}{4}$
          $\rule{0pt}{}$
           в—Ӣ $\sin(\theta)=\dfrac{рқ’ҡ}{3}$
          $\rule{0pt}{}$
     в—ү Use the Pythagorean identity:
          $\rule{0pt}{}$
          в—Ӣ $1=\cos^{2}(\theta)+\sin^{2}(\theta)$
          $\rule{0pt}{}$
     в—ү Substitute:
          $\rule{0pt}{}$
          в—Ӣ $1=\left(\dfrac{рқ’ҷ}{4}\right)^{2}+\left(\dfrac{рқ’ҡ}{3}\right)^{2}$
          $\rule{0pt}{}$
     в—ү Rectangular form:
          $\rule{0pt}{}$
          в—Ӣ $1=\dfrac{рқ’ҷ^{2}}{16}+\dfrac{рқ’ҡ^{2}}{9}$
          $\rule{0pt}{}$
     в—ү Conclusion:
          в—Ӣ the curve is an ellipse
     в—ү Geometry:
          в—Ӣ $x$-radius = 4
          в—Ӣ $y$-radius = 3


в—Ҹ Points and orientation for $\mathcal{I}=[0,2\pi]$.
     в—ү Initial Point:
          $\rule{0pt}{}$
          в—Ӣ $\theta=0$  вҶ’  $рқ’ҷ=4$, $рқ’ҡ=0$
          $\rule{0pt}{}$
          в—Ӣ so the initial point is $(4,0)$
     в—ү Midpoint checks:
          $\rule{0pt}{}$
          в—Ӣ $\theta=\dfrac{\pi}{2}$  вҶ’  $рқ’ҷ=0$, $рқ’ҡ=3$
          $\rule{0pt}{}$
          в—Ӣ $\theta=\pi$    вҶ’  $рқ’ҷ=-4$, $рқ’ҡ=0$
          $\rule{0pt}{}$
          в—Ӣ $\theta=\dfrac{3\pi}{2}$ вҶ’  $рқ’ҷ=0$, $рқ’ҡ=-3$
          $\rule{0pt}{}$
     в—ү Terminal Point:
          $\rule{0pt}{}$
          в—Ӣ $\theta=2\pi$  вҶ’  $рқ’ҷ=4$, $рқ’ҡ=0$
          $\rule{0pt}{}$
          в—Ӣ so the terminal point is $(4,0)$
     в—ү Orientation:
          в—Ӣ counterclockwise
     в—ү Final interpretation:
          в—Ӣ the object traces the full ellipse once
          в—Ӣ it starts and ends at $(4,0)$

в—Ҹ [1:17:13](https://www.youtube.com/watch?v=d4KADBFqpR0&t=4633). рҹ§© Example 6 вҖ” $рқ’ҷ=\sin(рқ‘»)$, $рқ’ҡ=\sin(2рқ‘»)$, with $\mathcal{I}=[0,2\pi]$.
     в—ү Use the double-angle identity:
     $\rule{0pt}{}$
          в—Ӣ $\sin(2рқ‘»)=2\sin(рқ‘»)\cos(рқ‘»)$
          $\rule{0pt}{}$
     в—ү Since $рқ’ҷ=\sin(рқ‘»)$,
     $\rule{0pt}{}$
          в—Ӣ $рқ’ҡ=2рқ’ҷ\cos(рқ‘»)$
          $\rule{0pt}{}$
     в—ү Solve for $\cos(рқ‘»)$:
     $\rule{0pt}{}$
          в—Ӣ $\cos(рқ‘»)=\dfrac{рқ’ҡ}{2рқ’ҷ}$
