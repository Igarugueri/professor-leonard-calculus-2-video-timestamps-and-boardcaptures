● [0:00]. Introduction to polar coordinates.
     ◉ Goal:
          ○ represent curves using angles and distances instead of rectangular coordinates
     ◉ Main idea:
          ○ polar graph paper uses concentric circles to measure distance from a central point
     ◉ Reason:
          ○ some curves are easier, nicer, or more relevant to describe without 𝒙 and 𝒚
     ◉ Important:
          ○ using polar graph paper helps visualize the angle and the directed distance directly


● [1:25]. Definition — The polar coordinate system.
     ◉ Polar axis:
          ○ the initial ray from which the angle starts
          ○ it plays the role of the positive x-axis
     ◉ Pole:
          ○ the central fixed point
          ○ this is the origin
     ◉ Main idea:
          ○ a point is described by an ordered pair (𝒓, 𝜽)
     ◉ Meaning:
          ○ 𝜽 gives the angle from the polar axis
          ○ 𝒓 gives the directed distance from the pole to the point


● [4:17]. Orientation of the angle 𝜽.
     ◉ Definition:
          ○ 𝜽 measures angular movement from the polar axis
     ◉ i̲f̲ 𝜽 > 0
          ○ t̲h̲e̲n̲ the direction is counterclockwise
     ◉ i̲f̲ 𝜽 < 0
          ○ t̲h̲e̲n̲ the direction is clockwise
     ◉ Important:
          ○ the sign of the angle determines the direction of rotation


● [4:53]. The directed distance 𝒓.
     ◉ Definition:
          ○ 𝒓 is the distance from the pole to the point
     ◉ i̲f̲ 𝒓 > 0
          ○ t̲h̲e̲n̲ the point lies on the terminal side of the angle
     ◉ i̲f̲ 𝒓 < 0
          ○ t̲h̲e̲n̲ use the given angle first
          ○ then reflect the point 180° through the pole
     ◉ Meaning:
          ○ negative 𝒓 sends the point in the opposite direction along the same line

● [16:53]. Coordinate conversion between polar and rectangular systems.
     ◉ Goal:
          ○ move back and forth between (𝒓, 𝜽) and (𝒙, 𝒚)
     ◉ Main idea:
          ○ use right-triangle trigonometry
     ◉ Polar to rectangular:
          ○ 𝒙 = 𝒓·cos(𝜽)
          ○ 𝒚 = 𝒓·sin(𝜽)
     ◉ Rectangular to polar:
          ○ 𝒓² = 𝒙² + 𝒚²
          ○ tan(𝜽) = 𝒚/𝒙
     ◉ Important:
          ○ always graph the point first
          ○ this prevents quadrant mistakes when choosing the angle
     ◉ Warning:
          ○ the square root in 𝒓 = √(𝒙² + 𝒚²) does not by itself fix the correct quadrant
          ○ the quadrant must be handled through the angle 𝜽

● [1:40:03]. Deriving 𝒅𝒚/𝒅𝜽.
     ◉ Start with:
          ○ 𝒚 = 𝒓 sin(𝜽)
     ◉ Differentiate with respect to 𝜽.
     ◉ Use the product rule:
          ○ 𝒅𝒚/𝒅𝜽 = 𝒅/𝒅𝜽 [𝒓] · sin(𝜽) + 𝒓 · 𝒅/𝒅𝜽 [sin(𝜽)]
     ◉ Therefore:
          ○ 𝒅𝒚/𝒅𝜽 = (𝒅𝒓/𝒅𝜽) sin(𝜽) + 𝒓 cos(𝜽)

● [1:44:04]. Deriving 𝒅𝒙/𝒅𝜽.
     ◉ Start with:
          ○ 𝒙 = 𝒓 cos(𝜽)
     ◉ Differentiate with respect to 𝜽.

● [1:59:36]. Horizontal and vertical tangents. 
     ◉ Horizontal tangents:+
          ○ i̲f̲ 𝒅𝒚/𝒅𝜽 = 0
               ■ t̲h̲e̲n̲ the slope is zero
               ■ so the tangent is horizontal
     ◉ Vertical tangents:
          ○ i̲f̲ 𝒅𝒙/𝒅𝜽 = 0
               ■ t̲h̲e̲n̲ the slope is undefined
               ■ so the tangent is vertical
     ◉ In terms of the formulas:
          ○ horizontal tangent when
               ■ (𝒅𝒓/𝒅𝜽) sin(𝜽) + 𝒓 cos(𝜽) = 0
          ○ vertical tangent when
               ■ (𝒅𝒓/𝒅𝜽) cos(𝜽) - 𝒓 sin(𝜽) = 0

● N̲O̲T̲E̲ 1 — Why the product rule is needed.
     ◉ in 𝒙 = 𝒓 cos(𝜽) and 𝒚 = 𝒓 sin(𝜽),
          ○ both factors depend on 𝜽
     ◉ cos(𝜽) and sin(𝜽) clearly depend on 𝜽
     ◉ and 𝒓 also depends on 𝜽 because the polar equation has the form 𝒓 = 𝒇(𝜽)
     ◉ so each derivative requires the product rule

● N̲O̲T̲E̲ 2 — Why 𝒅𝒓/𝒅𝜽 appears.
     ◉ 𝒓 is not treated as a constant
     ◉ it changes as 𝜽 changes
     ◉ so when differentiating 𝒓 with respect to 𝜽,
          ○ we get 𝒅𝒓/𝒅𝜽

● N̲O̲T̲E̲ 3 — Main workflow for polar slope problems.
     ◉ ❶ Write:
          ○ 𝒙 = 𝒓 cos(𝜽)
          ○ 𝒚 = 𝒓 sin(𝜽)
     ◉ ❷ Find 𝒅𝒓/𝒅𝜽.
     ◉ ❸ Compute:
          ○ 𝒅𝒚/𝒅𝜽 = (𝒅𝒓/𝒅𝜽) sin(𝜽) + 𝒓 cos(𝜽)
          ○ 𝒅𝒙/𝒅𝜽 = (𝒅𝒓/𝒅𝜽) cos(𝜽) - 𝒓 sin(𝜽)
     ◉ ❹ Form the ratio:
          ○ 𝒅𝒚/𝒅𝒙 = (𝒅𝒚/𝒅𝜽) / (𝒅𝒙/𝒅𝜽)
     ◉ ❺ Then evaluate at the requested angle
          ○ or set the numerator or denominator equal to zero for tangent-line questions

● [1:46:50]. 🧩 Example 6 — Find 𝒅𝒚/𝒅𝒙 and the slope of the tangent line for 𝒓 = 1 + cos(𝜽), at 𝜽 = π/6.
     ◉ ❶ Find 𝒅𝒓/𝒅𝜽 first.
          ○ 𝒓 = 1 + cos(𝜽)
          ○ 𝒅𝒓/𝒅𝜽 = -sin(𝜽)
     ◉ ❷ Use the polar slope formula.
          ○ 𝒅𝒚/𝒅𝒙 = [ (𝒅𝒓/𝒅𝜽) sin(𝜽) + 𝒓 cos(𝜽) ] / [ (𝒅𝒓/𝒅𝜽) cos(𝜽) - 𝒓 sin(𝜽) ]
     ◉ ❸ Substitute 𝒓 and 𝒅𝒓/𝒅𝜽.
          ○ 𝒅𝒚/𝒅𝒙 = [ (-sin(𝜽))sin(𝜽) + (1 + cos(𝜽))cos(𝜽) ] / [ (-sin(𝜽))cos(𝜽) - (1 + cos(𝜽))sin(𝜽) ]
     ◉ ❹ Expand and simplify.
          ○ numerator:
               ■ -sin²(𝜽) + cos(𝜽) + cos²(𝜽)
               ■ = cos²(𝜽) - sin²(𝜽) + cos(𝜽)
               ■ = cos(2𝜽) + cos(𝜽)
          ○ denominator:
               ■ -sin(𝜽)cos(𝜽) - sin(𝜽) - sin(𝜽)cos(𝜽)
               ■ = -2sin(𝜽)cos(𝜽) - sin(𝜽)
               ■ = -sin(2𝜽) - sin(𝜽)
     ◉ ❺ Therefore:
          ○ 𝒅𝒚/𝒅𝒙 = [ cos(2𝜽) + cos(𝜽) ] / [ -sin(2𝜽) - sin(𝜽) ]
          ○ 𝒅𝒚/𝒅𝒙 = - [ cos(2𝜽) + cos(𝜽) ] / [ sin(2𝜽) + sin(𝜽) ]
     ◉ ❻ Evaluate at 𝜽 = π/6.
          ○ 𝒎 = - [ cos(π/3) + cos(π/6) ] / [ sin(π/3) + sin(π/6) ]
          ○ 𝒎 = - [ 1/2 + √3/2 ] / [ √3/2 + 1/2 ]
          ○ 𝒎 = -1
     ◉ Final result:
          ○ 𝒅𝒚/𝒅𝒙 = - [ cos(2𝜽) + cos(𝜽) ] / [ sin(2𝜽) + sin(𝜽) ]
          ○ at 𝜽 = π/6, 𝒎 = -1
     ◉ Final interpretation:
          ○ the tangent line to the cardioid at 𝜽 = π/6 has slope -1

● N̲O̲T̲E̲ 4 — Horizontal and vertical tangents for polar curves.
     ◉ horizontal tangents happen when the main numerator is zero
          ○ that is, when 𝒅𝒚/𝒅𝜽 = 0
     ◉ vertical tangents happen when the main denominator is zero
          ○ that is, when 𝒅𝒙/𝒅𝜽 = 0
     ◉ Main idea:
          ○ this is exactly analogous to the parametric case
          ○ only now the parameter is 𝜽 instead of 𝒕
