в—Ҹ [1:03]. Derivation of the Integration by Parts formula from the Product Rule. 
     в—ү Starting point: the Product Rule.

     в—ү [5:29]. Introduction of auxiliary notation.

     в—ү Common mistake: choosing рқ’– = рқ’ҷВі.

     в—ү Main obstacle:
          в—Ӣ ln рқ’ҷ is difficult to integrate if chosen as рқ’…рқ“Ҙ.
     в—ү Correct choice:
          в—Ӣ рқ’– = ln рқ’ҷ

          в—Ӣ рқ’…рқ’– = 1/рқ’ҷ рқ’…рқ’ҷ

          в—Ӣ вҲ« рқ’ҷВі ln рқ’ҷ рқ’…рқ’ҷ = рқ’–рқ“Ҙ вҲ’ вҲ« рқ“Ҙ рқ’…рқ’–
          в—Ӣ = ln рқ’ҷ В· рқ’ҷвҒҙ/4 вҲ’ вҲ« (рқ’ҷвҒҙ/4)(1/рқ’ҷ) рқ’…рқ’ҷ
     в—ү Simplify the remaining integral:
          в—Ӣ = рқ’ҷвҒҙ/4 ln рқ’ҷ вҲ’ (1/4) вҲ« рқ’ҷВі рқ’…рқ’ҷ
          в—Ӣ = рқ’ҷвҒҙ/4 ln рқ’ҷ вҲ’ (1/4)(рқ’ҷвҒҙ/4)
     в—ү Final result:
          в—Ӣ вҲ« рқ’ҷВі ln рқ’ҷ рқ’…рқ’ҷ = (1/4)рқ’ҷвҒҙ ln рқ’ҷ вҲ’ (1/16)рқ’ҷвҒҙ + рқ“’

в—Ҹ [28:42]. рҹ§© Example 3 вҖ” Repeated use of Integration by Parts: Evaluate вҲ« рқ’ҷВІ sin(2рқ’ҷ) рқ’…рқ’ҷ 
     в—ү First iteration of Integration by Parts.
          в—Ӣ Choose:
               в–  рқ’– = рқ’ҷВІ
               в–  рқ’…рқ“Ҙ = sin(2рқ’ҷ) рқ’…рқ’ҷ
          в—Ӣ Then:
               в–  рқ’…рқ’– = 2рқ’ҷ рқ’…рқ’ҷ
               в–  рқ“Ҙ = вҲ« sin(2рқ’ҷ) рқ’…рқ’ҷ = вҲ’(1/2) cos(2рқ’ҷ)
     в—ү Apply the formula.
          в—Ӣ вҲ« рқ’ҷВІ sin(2рқ’ҷ) рқ’…рқ’ҷ = рқ’–рқ“Ҙ вҲ’ вҲ« рқ“Ҙ рқ’…рқ’–
          в—Ӣ = рқ’ҷВІ [вҲ’(1/2) cos(2рқ’ҷ)] вҲ’ вҲ« [вҲ’(1/2) cos(2рқ’ҷ)] (2рқ’ҷ) рқ’…рқ’ҷ
          в—Ӣ = вҲ’(1/2)рқ’ҷВІ cos(2рқ’ҷ) + вҲ« рқ’ҷ cos(2рқ’ҷ) рқ’…рқ’ҷ
     в—ү Second iteration on the remaining integral.
          в—Ӣ For вҲ« рқ’ҷ cos(2рқ’ҷ) рқ’…рқ’ҷ, choose:
               в–  рқ’– = рқ’ҷ
               в–  рқ’…рқ“Ҙ = cos(2рқ’ҷ) рқ’…рқ’ҷ
          в—Ӣ Then:
               в–  рқ’…рқ’– = рқ’…рқ’ҷ
               в–  рқ“Ҙ = вҲ« cos(2рқ’ҷ) рқ’…рқ’ҷ = (1/2) sin(2рқ’ҷ)
     в—ү Apply Integration by Parts again.
          в—Ӣ вҲ« рқ’ҷ cos(2рқ’ҷ) рқ’…рқ’ҷ = рқ’–рқ“Ҙ вҲ’ вҲ« рқ“Ҙ рқ’…рқ’–
          в—Ӣ = рқ’ҷ[(1/2) sin(2рқ’ҷ)] вҲ’ вҲ« (1/2) sin(2рқ’ҷ) рқ’…рқ’ҷ
          в—Ӣ = (1/2)рқ’ҷ sin(2рқ’ҷ) вҲ’ (1/2) вҲ« sin(2рқ’ҷ) рқ’…рқ’ҷ
          в—Ӣ = (1/2)рқ’ҷ sin(2рқ’ҷ) + (1/4) cos(2рқ’ҷ)
     в—ү Consolidate all terms.
          в—Ӣ вҲ« рқ’ҷВІ sin(2рқ’ҷ) рқ’…рқ’ҷ
               в–  = вҲ’(1/2)рқ’ҷВІ cos(2рқ’ҷ) + (1/2)рқ’ҷ sin(2рқ’ҷ) + (1/4) cos(2рқ’ҷ) + рқ“’
     в—ү Key idea.
          в—Ӣ Each application of Integration by Parts lowers the power of рқ’ҷ.
          в—Ӣ Repeating the method eventually reduces the problem to a basic trigonometric integral.

в—Ҹ [43:02]. рҹ§© Example 4 вҖ” Circula integral: вҲ« рқ’ҶЛЈ sin(2рқ’ҷ) рқ’…рқ’ҷ. 
     в—ү Observation of the cyclic behavior of transcendental functions.
          в—Ӣ Repeated integration by parts brings back the original integral.
          в—Ӣ This creates a circular equation that must be solved algebraically.
     в—ү First application of Integration by Parts.
          в—Ӣ Choose:
               в–  рқ’– = sin(2рқ’ҷ)
               в–  рқ’…рқ“Ҙ = рқ’ҶЛЈ рқ’…рқ’ҷ
          в—Ӣ Then:

          в—Ӣ Apply the formula:
               в–  вҲ« рқ’ҶЛЈ sin(2рқ’ҷ) рқ’…рқ’ҷ = рқ’ҶЛЈ sin(2рқ’ҷ) вҲ’ 2вҲ« рқ’ҶЛЈ cos(2рқ’ҷ) рқ’…рқ’ҷ
     в—ү Second application on the new integral.
          в—Ӣ For вҲ« рқ’ҶЛЈ cos(2рқ’ҷ) рқ’…рқ’ҷ, choose:
               в–  рқ’– = cos(2рқ’ҷ)
               в–  рқ’…рқ“Ҙ = рқ’ҶЛЈ рқ’…рқ’ҷ
          в—Ӣ Then:
               в–  рқ’…рқ’– = вҲ’2 sin(2рқ’ҷ) рқ’…рқ’ҷ
               в–  рқ“Ҙ = рқ’ҶЛЈ
          в—Ӣ Apply the formula:
               в–  вҲ« рқ’ҶЛЈ cos(2рқ’ҷ) рқ’…рқ’ҷ = рқ’ҶЛЈ cos(2рқ’ҷ) + 2вҲ« рқ’ҶЛЈ sin(2рқ’ҷ) рқ’…рқ’ҷ
     в—ү Substitute back into the first result.
          в—Ӣ вҲ« рқ’ҶЛЈ sin(2рқ’ҷ) рқ’…рқ’ҷ
               в–  = рқ’ҶЛЈ sin(2рқ’ҷ) вҲ’ 2[рқ’ҶЛЈ cos(2рқ’ҷ) + 2вҲ« рқ’ҶЛЈ sin(2рқ’ҷ) рқ’…рқ’ҷ]
               в–  = рқ’ҶЛЈ sin(2рқ’ҷ) вҲ’ 2рқ’ҶЛЈ cos(2рқ’ҷ) вҲ’ 4вҲ« рқ’ҶЛЈ sin(2рқ’ҷ) рқ’…рқ’ҷ
     в—ү Solve algebraically by isolating the original integral.
          в—Ӣ Add 4вҲ« рқ’ҶЛЈ sin(2рқ’ҷ) рқ’…рқ’ҷ to both sides:
               в–  5вҲ« рқ’ҶЛЈ sin(2рқ’ҷ) рқ’…рқ’ҷ = рқ’ҶЛЈ sin(2рқ’ҷ) вҲ’ 2рқ’ҶЛЈ cos(2рқ’ҷ)
          в—Ӣ Divide by 5:
               в–  вҲ« рқ’ҶЛЈ sin(2рқ’ҷ) рқ’…рқ’ҷ = (1/5)рқ’ҶЛЈ[sin(2рқ’ҷ) вҲ’ 2 cos(2рқ’ҷ)] + рқ“’
     в—ү Final result:
          в—Ӣ вҲ« рқ’ҶЛЈ sin(2рқ’ҷ) рқ’…рқ’ҷ = (1/5)рқ’ҶЛЈ[sin(2рқ’ҷ) вҲ’ 2 cos(2рқ’ҷ)] + рқ“’

в—Ҹ [59:00]. рҹ§© Example 5 вҖ” Definite integral and geometric interpretation: вҲ«[1, рқ’Ҷ] ln рқ’ҷ рқ’…рқ’ҷ. 
     в—ү Define the parts.
          в—Ӣ рқ’– = ln рқ’ҷ
          в—Ӣ рқ’…рқ“Ҙ = рқ’…рқ’ҷ
          в—Ӣ рқ’…рқ’– = (1/рқ’ҷ) рқ’…рқ’ҷ
          в—Ӣ рқ“Ҙ = рқ’ҷ
     в—ү Apply Integration by Parts.

          в—Ӣ = рқ’ҷ ln рқ’ҷ вҲ’ вҲ« рқ’ҷ В· (1/рқ’ҷ) рқ’…рқ’ҷ
          в—Ӣ = рқ’ҷ ln рқ’ҷ вҲ’ вҲ« 1 рқ’…рқ’ҷ
     в—ү Antiderivative obtained.
          в—Ӣ рқ’ҷ ln рқ’ҷ вҲ’ рқ’ҷ
     в—ү Evaluate at the bounds.

     в—ү Geometric interpretation.
          в—Ӣ The definite integral represents the area under the curve рқ’ҡ = ln рқ’ҷ from рқ’ҷ = 1 to рқ’ҷ = рқ’Ҷ.
     в—ү Final conclusion.
          в—Ӣ вҲ«[1, рқ’Ҷ] ln рқ’ҷ рқ’…рқ’ҷ = 1
          в—Ӣ Therefore, the area under ln рқ’ҷ on [1, рқ’Ҷ] is exactly 1.

               в–  вҲ« secвҒө рқ’ҷ рқ’…рқ’ҷ = secВі рқ’ҷ tan рқ’ҷ вҲ’ 3вҲ« tanВІ рқ’ҷ secВі рқ’ҷ рқ’…рқ’ҷ
          в—Ӣ Use tanВІ рқ’ҷ = secВІ рқ’ҷ вҲ’ 1:
               в–  вҲ« secвҒө рқ’ҷ рқ’…рқ’ҷ = secВі рқ’ҷ tan рқ’ҷ вҲ’ 3вҲ« (secВІ рқ’ҷ вҲ’ 1)secВі рқ’ҷ рқ’…рқ’ҷ
               в–  = secВі рқ’ҷ tan рқ’ҷ вҲ’ 3вҲ« secвҒө рқ’ҷ рқ’…рқ’ҷ + 3вҲ« secВі рқ’ҷ рқ’…рқ’ҷ
          в—Ӣ Solve for the original integral:
               в–  4вҲ« secвҒө рқ’ҷ рқ’…рқ’ҷ = secВі рқ’ҷ tan рқ’ҷ + 3вҲ« secВі рқ’ҷ рқ’…рқ’ҷ
          в—Ӣ Substitute the previous result for вҲ« secВі рқ’ҷ рқ’…рқ’ҷ:
               в–  4вҲ« secвҒө рқ’ҷ рқ’…рқ’ҷ = secВі рқ’ҷ tan рқ’ҷ + (3/2)sec рқ’ҷ tan рқ’ҷ + (3/2)ln|sec рқ’ҷ + tan рқ’ҷ|
          в—Ӣ Final result:
               в–  вҲ« secвҒө рқ’ҷ рқ’…рқ’ҷ = (1/4)secВі рқ’ҷ tan рқ’ҷ + (3/8)sec рқ’ҷ tan рқ’ҷ + (3/8)ln|sec рқ’ҷ + tan рқ’ҷ| + рқ“’
     в—ү Key idea
          в—Ӣ Odd powers of secant are handled by peeling off a factor secВІ рқ’ҷ and using Integration by Parts.
          в—Ӣ The identity tanВІ рқ’ҷ = secВІ рқ’ҷ вҲ’ 1 converts the remaining expression back into secant powers, creating a recurrence relation.

в—Ҹ [1:29:40]. Development and proof of the Reduction Formula. 
     в—ү NOTE: It is not only for sine. This is specifically the reduction formula for вҲ« sinвҒҝx dx, but similar reduction formulas also exist for cosine, secant, and other families of integrals.
     в—ү Goal:
          в—Ӣ Reduce вҲ« sinвҒҝ рқ’ҷ рқ’…рқ’ҷ to an integral involving sinвҒҝвҒ»ВІ рқ’ҷ.
          в—Ӣ Assume рқ’Ҹ вүҘ 2.
     в—ү Split the integrand:
          в—Ӣ вҲ« sinвҒҝ рқ’ҷ рқ’…рқ’ҷ = вҲ« sinвҒҝвҒ»В№ рқ’ҷ В· sin рқ’ҷ рқ’…рқ’ҷ
     в—ү Apply Integration by Parts.
          в—Ӣ Choose:
               в–  рқ’– = sinвҒҝвҒ»В№ рқ’ҷ
               в–  рқ’…рқ“Ҙ = sin рқ’ҷ рқ’…рқ’ҷ
          в—Ӣ Then:
               в–  рқ’…рқ’– = (рқ’ҸвҲ’1) sinвҒҝвҒ»ВІ рқ’ҷ cos рқ’ҷ рқ’…рқ’ҷ
               в–  рқ“Ҙ = вҲ’cos рқ’ҷ
     в—ү Substitute into the formula.
          в—Ӣ вҲ« sinвҒҝ рқ’ҷ рқ’…рқ’ҷ
               в–  = рқ’–рқ“Ҙ вҲ’ вҲ« рқ“Ҙ рқ’…рқ’–
               в–  = вҲ’sinвҒҝвҒ»В№ рқ’ҷ cos рқ’ҷ + (рқ’ҸвҲ’1)вҲ« cosВІ рқ’ҷ sinвҒҝвҒ»ВІ рқ’ҷ рқ’…рқ’ҷ
     в—ү Rewrite cosВІ рқ’ҷ in terms of sine.
          в—Ӣ Use:
               в–  cosВІ рқ’ҷ = 1 вҲ’ sinВІ рқ’ҷ
          в—Ӣ Then:
               в–  вҲ« sinвҒҝ рқ’ҷ рқ’…рқ’ҷ
               в–  = вҲ’sinвҒҝвҒ»В№ рқ’ҷ cos рқ’ҷ + (рқ’ҸвҲ’1)вҲ« (1 вҲ’ sinВІ рқ’ҷ) sinвҒҝвҒ»ВІ рқ’ҷ рқ’…рқ’ҷ
     в—ү Expand the integral.
          в—Ӣ = вҲ’sinвҒҝвҒ»В№ рқ’ҷ cos рқ’ҷ + (рқ’ҸвҲ’1)вҲ« sinвҒҝвҒ»ВІ рқ’ҷ рқ’…рқ’ҷ вҲ’ (рқ’ҸвҲ’1)вҲ« sinвҒҝ рқ’ҷ рқ’…рқ’ҷ
     в—ү Collect like terms.
          в—Ӣ Add (рқ’ҸвҲ’1)вҲ« sinвҒҝ рқ’ҷ рқ’…рқ’ҷ to both sides:
               в–  рқ’ҸвҲ« sinвҒҝ рқ’ҷ рқ’…рқ’ҷ = вҲ’sinвҒҝвҒ»В№ рқ’ҷ cos рқ’ҷ + (рқ’ҸвҲ’1)вҲ« sinвҒҝвҒ»ВІ рқ’ҷ рқ’…рқ’ҷ
     в—ү Divide by рқ’Ҹ.
          в—Ӣ вҲ« sinвҒҝ рқ’ҷ рқ’…рқ’ҷ = вҲ’(1/рқ’Ҹ) sinвҒҝвҒ»В№ рқ’ҷ cos рқ’ҷ + ((рқ’ҸвҲ’1)/рқ’Ҹ)вҲ« sinвҒҝвҒ»ВІ рқ’ҷ рқ’…рқ’ҷ
     в—ү Final Reduction Formula for sine:
          в—Ӣ вҲ« sinвҒҝ рқ’ҷ рқ’…рқ’ҷ = вҲ’(1/рқ’Ҹ) sinвҒҝвҒ»В№ рқ’ҷ cos рқ’ҷ + ((рқ’ҸвҲ’1)/рқ’Ҹ)вҲ« sinвҒҝвҒ»ВІ рқ’ҷ рқ’…рқ’ҷ
          
в—Ҹ  [1:48:13]. рҹ§© Example вҖ”: вҲ« sinвҒҙ рқ’ҷ рқ’…рқ’ҷ. 
     в—ү Apply the Reduction Formula with рқ’Ҹ = 4.
          в—Ӣ вҲ« sinвҒҙ рқ’ҷ рқ’…рқ’ҷ = вҲ’(1/4) sinВі рқ’ҷ cos рқ’ҷ + (3/4) вҲ« sinВІ рқ’ҷ рқ’…рқ’ҷ
     в—ү Reduce the remaining integral вҲ« sinВІ рқ’ҷ рқ’…рқ’ҷ.
          в—Ӣ Apply the Reduction Formula again with рқ’Ҹ = 2:
               в–  вҲ« sinВІ рқ’ҷ рқ’…рқ’ҷ = вҲ’(1/2) sin рқ’ҷ cos рқ’ҷ + (1/2) вҲ« 1 рқ’…рқ’ҷ
               в–  = вҲ’(1/2) sin рқ’ҷ cos рқ’ҷ + (1/2)рқ’ҷ
     в—ү Substitute this result back into the first expression.
          в—Ӣ вҲ« sinвҒҙ рқ’ҷ рқ’…рқ’ҷ
               в–  = вҲ’(1/4) sinВі рқ’ҷ cos рқ’ҷ + (3/4)[вҲ’(1/2) sin рқ’ҷ cos рқ’ҷ + (1/2)рқ’ҷ]
     в—ү Distribute the factor 3/4.
          в—Ӣ вҲ« sinвҒҙ рқ’ҷ рқ’…рқ’ҷ
               в–  = вҲ’(1/4) sinВі рқ’ҷ cos рқ’ҷ вҲ’ (3/8) sin рқ’ҷ cos рқ’ҷ + (3/8)рқ’ҷ + рқ“’
     в—ү Final result.
          в—Ӣ вҲ« sinвҒҙ рқ’ҷ рқ’…рқ’ҷ = вҲ’(1/4) sinВі рқ’ҷ cos рқ’ҷ вҲ’ (3/8) sin рқ’ҷ cos рқ’ҷ + (3/8)рқ’ҷ + рқ“’
