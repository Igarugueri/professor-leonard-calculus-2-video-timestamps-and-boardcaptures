в—Ҹ [0:01]. Introduction to inverse trigonometric functions and fundamental properties.
     в—ү One-to-one requirement: a function must be injective to have an inverse.
          в—Ӣ A function is injective if different inputs always produce different outputs.
               в–  This guarantees that each value in the range corresponds to exactly one value in the domain.
               в–  Without injectivity, an inverse function would be ambiguous (one output mapping to multiple inputs).
          в—Ӣ Need for domain restriction in periodic trigonometric functions.
               в–  Trigonometric functions like sin, cos, and tan are periodic and repeat values infinitely many times.
               в–  Restricting the domain removes this repetition and forces the function to be one-to-one.
               в–  The chosen restricted interval is called the principal interval and defines the principal values of the inverse.

     в—ү Key graphical properties.
          в—Ӣ The sine function is periodic and oscillates between вҲ’1 and 1.
          в—Ӣ It fails the **horizontal line test** on в„қ, so it is not one-to-one on its full domain.
     в—ү Restricting the domain to the interval [вҲ’ПҖ/2, ПҖ/2] to ensure injectivity.
          в—Ӣ We restrict the domain so that the inverse is a function.
          в—Ӣ On this interval, sin(рқ’ҷ) is strictly increasing.
          в—Ӣ Every horizontal line intersects the graph at most once.
