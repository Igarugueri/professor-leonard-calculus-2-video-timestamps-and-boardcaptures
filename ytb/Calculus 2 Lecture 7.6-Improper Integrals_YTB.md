в—Ҹ [2:14]. Classification of the two fundamental cases. 
     в—ү Informal idea:
          в—Ӣ An improper integral appears when the usual conditions for a definite integral fail.
          в—Ӣ That typically happens because:
               в–  the interval is infinite,
               в–  or the function is unbounded on the interval.
     в—ү **Case 1**:
          в—Ӣ The interval of integration is infinite.
          в—Ӣ It starts at рқ’Ӯ but extends without bound.
     в—ү **Case 2**:
          в—Ӣ The interval [рқ’Ӯ,рқ’ғ] is finite,
          в—Ӣ but the function рқ’Ү(рқ’ҷ) is unbounded (has an infinite discontinuity) at an endpoint or at some point inside the interval.

в—Ҹ [8:12]. рҹ§© Example 1 вҖ” Evaluate the integral вҲ«[1, вҲһ] 1/рқ’ҷВІ рқ’…рқ’ҷ. 
     в—ү 1. Identify why the integral is improper.
          в—Ӣ The function 1/рқ’ҷВІ is defined for every рқ’ҷ вүҘ 1.
          в—Ӣ There is no infinite discontinuity on [1, вҲһ).
          в—Ӣ The improper behavior comes from the interval extending to infinity.
          в—Ӣ So this is **Case 1**: an infinite interval.
          в—Ӣ But if we changed the interval to [вҲ’1, вҲһ], then at рқ’ҷ = 0 the function would be undefined.
               в–  It would have an infinite discontinuity there.
               в–  In that situation, a vertical asymptote appears.
               в–  So **Case 2** would also be present.
     в—ү 2. Replace the infinite upper limit with a variable.
          в—Ӣ Consider вҲ«[1, рқ’ғ] 1/рқ’ҷВІ рқ’…рқ’ҷ, where рқ’ғ > 1.
          в—Ӣ We do this because we cannot substitute вҲһ directly into the integral.
     в—ү 3. Visual interpretation.
          в—Ӣ This represents the area under 1/рқ’ҷВІ from рқ’ҷ = 1 to рқ’ҷ = рқ’ғ.
          в—Ӣ Then we study what happens as рқ’ғ grows without bound.
     в—ү 4. Evaluate the finite integral first.
          в—Ӣ вҲ«[1, рқ’ғ] 1/рқ’ҷВІ рқ’…рқ’ҷ
          в—Ӣ = вҲ«[1, рқ’ғ] рқ’ҷвҒ»ВІ рқ’…рқ’ҷ
          в—Ӣ = [рқ’ҷвҒ»В№ / (вҲ’1)]_[1,рқ’ғ]
          в—Ӣ = [вҲ’1/рқ’ҷ]_[1,рқ’ғ]
          в—Ӣ = вҲ’1/рқ’ғ вҲ’ (вҲ’1/1)
          в—Ӣ = вҲ’1/рқ’ғ + 1
     в—ү [15:53]. Use a limit to recover the improper integral.
          в—Ӣ The way we handle вҖңplugging in infinityвҖқ is with a limit.
          в—Ӣ lim_(рқ’ғ вҶ’ вҲһ) вҲ«[1, рқ’ғ] 1/рқ’ҷВІ рқ’…рқ’ҷ
          в—Ӣ = lim_(рқ’ғ вҶ’ вҲһ) (вҲ’1/рқ’ғ + 1)
          в—Ӣ = 0 + 1
          в—Ӣ = 1
     в—ү Final evaluation.
          в—Ӣ The integral converges.
          в—Ӣ вҲ«[1, вҲһ] 1/рқ’ҷВІ рқ’…рқ’ҷ = 1

в—Ҹ [18:34]. General notation and definitions of convergence. 
     в—ү Formal notation:
          в—Ӣ вҲ«[рқ’Ӯ, вҲһ] рқ’Ү(рқ’ҷ) вӢ… рқ’…рқ’ҷ = lim_(рқ’ғ вҶ’ вҲһ) вҲ«[рқ’Ӯ, рқ’ғ] рқ’Ү(рқ’ҷ) вӢ… рқ’…рқ’ҷ
          в—Ӣ вҲ«[вҲ’вҲһ, рқ’ғ] рқ’Ү(рқ’ҷ) вӢ… рқ’…рқ’ҷ = lim_(рқ’Ӯ вҶ’ вҲ’вҲһ) вҲ«[рқ’Ӯ, рқ’ғ] рқ’Ү(рқ’ҷ) вӢ… рқ’…рқ’ҷ
     в—ү [21:55]. Definitions:
          в—Ӣ **Convergence**:
               в–  the limit exists and is finite
          в—Ӣ **Divergence**:
               в–  the limit does not exist, or is infinite

          в—Ӣ The improper behavior comes from the interval extending to infinity.
          в—Ӣ So this is **Case 1**: an infinite interval.
     в—ү Replace the infinite upper limit with a variable.

     в—ү Rewrite the improper integral as a limit.

в—Ҹ [31:10]. рҹ§© Example 3 вҖ” **Generalization for power integrals (p-integrals)**: вҲ«[1, вҲһ] 1/рқ’ҷвҒҝ вӢ… рқ’…рқ’ҷ. 
     в—ү Goal:
          в—Ӣ Study the general behavior of improper integrals of the form вҲ«[1, вҲһ] 1/рқ’ҷвҒҝ вӢ… рқ’…рқ’ҷ.
     в—ү Rewrite the improper integral as a limit.
          в—Ӣ вҲ«[1, вҲһ] 1/рқ’ҷвҒҝ вӢ… рқ’…рқ’ҷ = lim_(рқ’ғ вҶ’ вҲһ) вҲ«[1, рқ’ғ] 1/рқ’ҷвҒҝ вӢ… рқ’…рқ’ҷ
          в—Ӣ = lim_(рқ’ғ вҶ’ вҲһ) вҲ«[1, рқ’ғ] рқ’ҷвҒ»вҒҝ вӢ… рқ’…рқ’ҷ
     в—ү Important precondition.
          в—Ӣ The formal antiderivative used below is valid only for рқ’Ҹ вү  1.
          в—Ӣ So the case рқ’Ҹ = 1 must be treated separately.
          в—Ӣ If рқ’Ҹ = 1, then
               в–  вҲ« 1/рқ’ҷ рқ’…рқ’ҷ = ln|рқ’ҷ|
     в—ү Integrate formally for рқ’Ҹ вү  1.
          в—Ӣ lim_(рқ’ғ вҶ’ вҲһ) [рқ’ҷвҒ»вҒҝвҒәВ№ / (вҲ’рқ’Ҹ + 1)] | [1, рқ’ғ]
          в—Ӣ = lim_(рқ’ғ вҶ’ вҲһ) [рқ’ғвҒ»вҒҝвҒәВ№ / (1 вҲ’ рқ’Ҹ) вҲ’ 1 / (1 вҲ’ рқ’Ҹ)]
          в—Ӣ = lim_(рқ’ғ вҶ’ вҲһ) (рқ’ғвҒ»вҒҝвҒәВ№ вҲ’ 1) / (1 вҲ’ рқ’Ҹ)
     в—ү Rewrite the exponent to study the limit.
          в—Ӣ вҲ’рқ’Ҹ + 1 = вҲ’(рқ’Ҹ вҲ’ 1)
          в—Ӣ so рқ’ғвҒ»вҒҝвҒәВ№ = 1 / рқ’ғвҒҝвҒ»В№
          в—Ӣ Hence:
               в–  вҲ«[1, вҲһ] 1/рқ’ҷвҒҝ вӢ… рқ’…рқ’ҷ = 1 / (1 вҲ’ рқ’Ҹ) вӢ… lim_(рқ’ғ вҶ’ вҲһ) [1 / рқ’ғвҒҝвҒ»В№ вҲ’ 1]
     в—ү Analyze the limit.
          в—Ӣ If рқ’Ҹ > 1,
               в–  then рқ’Ҹ вҲ’ 1 > 0
               в–  so 1 / рқ’ғвҒҝвҒ»В№ вҶ’ 0
               в–  and the integral converges.
          в—Ӣ If рқ’Ҹ = 1,
               в–  the integral becomes вҲ«[1, вҲһ] 1/рқ’ҷ вӢ… рқ’…рқ’ҷ
               в–  which diverges, as shown in the previous example.
          в—Ӣ If рқ’Ҹ < 1,
               в–  then рқ’Ҹ вҲ’ 1 < 0
               в–  so рқ’ғвҒ»вҒҝвҒәВ№ grows without bound
               в–  and the integral diverges.
     в—ү **Final p-integral criterion.**
          в—Ӣ вҲ«[1, вҲһ] 1/рқ’ҷвҒҝ вӢ… рқ’…рқ’ҷ **converges if рқ’Ҹ > 1**
          в—Ӣ вҲ«[1, вҲһ] 1/рқ’ҷвҒҝ вӢ… рқ’…рқ’ҷ **diverges if рқ’Ҹ вүӨ 1**      


в—Ҹ [43:46]. рҹ§© Example 4 вҖ” Evaluate the improper integral вҲ«[1, вҲһ] eвҒ»ВІрқ’ҷ вӢ… рқ’…рқ’ҷ.

в—Ҹ [1:00:30]. рҹ§© Example 6 вҖ” Evaluate the improper integral вҲ«[-вҲһ, 0] рқ’ҷeЛЈ вӢ… рқ’…рқ’ҷ. 
     в—ү Identify why the integral is improper.

          в—Ӣ The interval extends to вҲ’вҲһ.
          в—Ӣ There is no infinite discontinuity in the integrand рқ’ҷeЛЈ on (вҲ’вҲһ, 0].
     в—ү Rewrite the improper integral as a limit.
          в—Ӣ вҲ«[-вҲһ, 0] рқ’ҷeЛЈ вӢ… рқ’…рқ’ҷ = lim_(рқ’Ӯ вҶ’ вҲ’вҲһ) вҲ«[рқ’Ӯ, 0] рқ’ҷeЛЈ вӢ… рқ’…рқ’ҷ

               в–  рқ’…рқ“Ҙ = eЛЈ рқ’…рқ’ҷ
          в—Ӣ Then:

в—Ҹ [1:17:15]. рҹ§© Example 7 вҖ” **Integrals over the entire real line**: вҲ«[-вҲһ, вҲһ] eЛЈ/(1 + eВІЛЈ) вӢ… рқ’…рқ’ҷ 
     в—ү 1. Identify why the integral is improper.
          в—Ӣ The function eЛЈ/(1 + eВІЛЈ) is defined everywhere.
          в—Ӣ There is no infinite discontinuity on [-вҲһ, вҲһ].
          в—Ӣ The improper behavior comes from the interval extending to infinity.
          в—Ӣ So this is **Case 1**: an infinite interval.
     в—ү General form:
          в—Ӣ вҲ«[-вҲһ, вҲһ] рқ’Ү(рқ’ҷ) вӢ… рқ’…рқ’ҷ
     в—ү [1:20:03]. Additivity property:
          в—Ӣ Split the integral at рқ’ҷ = 0.
          в—Ӣ вҲ«[-вҲһ, вҲһ] eЛЈ/(1 + eВІЛЈ) вӢ… рқ’…рқ’ҷ
          в—Ӣ = вҲ«[-вҲһ, 0] eЛЈ/(1 + eВІЛЈ) вӢ… рқ’…рқ’ҷ + вҲ«[0, вҲһ] eЛЈ/(1 + eВІЛЈ) вӢ… рқ’…рқ’ҷ
     в—ү Rewrite both pieces as limits.
          в—Ӣ lim_(рқ’Ӯ вҶ’ вҲ’вҲһ) вҲ«[рқ’Ӯ, 0] eЛЈ/(1 + eВІЛЈ) вӢ… рқ’…рқ’ҷ
          в—Ӣ + lim_(рқ’ғ вҶ’ вҲһ) вҲ«[0, рқ’ғ] eЛЈ/(1 + eВІЛЈ) вӢ… рқ’…рқ’ҷ
     в—ү Substitution for the integrand.
          в—Ӣ Let рқ’– = eЛЈ
          в—Ӣ Then рқ’…рқ’– = eЛЈ рқ’…рқ’ҷ
          в—Ӣ So:
               в–  вҲ« eЛЈ/(1 + eВІЛЈ) вӢ… рқ’…рқ’ҷ = вҲ« 1/(1 + рқ’–ВІ) вӢ… рқ’…рқ’–
               в–  = tanвҒ»В№(рқ’–)
               в–  = tanвҒ»В№(eЛЈ)
     в—ү Apply the antiderivative to both improper pieces.
          в—Ӣ lim_(рқ’Ӯ вҶ’ вҲ’вҲһ) [tanвҒ»В№(eЛЈ)] | [рқ’Ӯ,0]
          в—Ӣ + lim_(рқ’ғ вҶ’ вҲһ) [tanвҒ»В№(eЛЈ)] | [0,рқ’ғ]
     в—ү Evaluate each limit.
          в—Ӣ First piece:
               в–  lim_(рқ’Ӯ вҶ’ вҲ’вҲһ) [tanвҒ»В№(eвҒ°) вҲ’ tanвҒ»В№(eбөғ)]
               в–  = tanвҒ»В№(1) вҲ’ tanвҒ»В№(0)
               в–  = ПҖ/4 вҲ’ 0

в—Ҹ [2:27:00]. Comparison Theorem for improper integrals. 
     в—ү Setup:
          в—Ӣ Assume 0 вүӨ рқ’Ҳ(рқ’ҷ) вүӨ рқ’Ү(рқ’ҷ) on [рқ’Ӯ, вҲһ).
          в—Ӣ So рқ’Ү(рқ’ҷ) is above рқ’Ҳ(рқ’ҷ) on the interval.
     в—ү Fundamental comparison:
          в—Ӣ If вҲ«[рқ’Ӯ, вҲһ] рқ’Ү(рқ’ҷ) вӢ… рқ’…рқ’ҷ converges,
          в—Ӣ then вҲ«[рқ’Ӯ, вҲһ] рқ’Ҳ(рқ’ҷ) вӢ… рқ’…рқ’ҷ also converges.
          в—Ӣ Intuition:
               в–  if the bigger area is finite,
               в–  then the smaller area must also be finite.
     в—ү Reverse comparison:
          в—Ӣ If вҲ«[рқ’Ӯ, вҲһ] рқ’Ҳ(рқ’ҷ) вӢ… рқ’…рқ’ҷ diverges,
          в—Ӣ then вҲ«[рқ’Ӯ, вҲһ] рқ’Ү(рқ’ҷ) вӢ… рқ’…рқ’ҷ also diverges.
          в—Ӣ Intuition:
               в–  if the smaller area is already infinite,
               в–  then the bigger area must also be infinite.
     в—ү Important conditions:
          в—Ӣ The comparison is stated for nonnegative functions.
          в—Ӣ That is why we require 0 вүӨ рқ’Ҳ(рқ’ҷ) вүӨ рқ’Ү(рқ’ҷ).
     в—ү Important warning:
          в—Ӣ The implications only work in these directions.
          в—Ӣ You may conclude:
               в–  bigger converges вҮ’ smaller converges
               в–  smaller diverges вҮ’ bigger diverges
          в—Ӣ But you may not conclude:
               в–  smaller converges вҮ’ bigger converges
               в–  bigger diverges вҮ’ smaller diverges
     в—ү Main idea:
          в—Ӣ Instead of integrating a difficult function directly,
          в—Ӣ compare it with a simpler function whose improper behavior is already known.

     в—ү Key inequality:
          в—Ӣ Since sin(рқ’ҷ) вүӨ 1,
               в–  then sinВІ(рқ’ҷ) вүӨ 1
          в—Ӣ Adding рқ’ҷ to both sides gives:
               в–  рқ’ҷ + sinВІ(рқ’ҷ) вүӨ рқ’ҷ + 1
          в—Ӣ Taking reciprocals reverses the inequality:
               в–  1/(рқ’ҷ + sinВІ рқ’ҷ) вүҘ 1/(1 + рқ’ҷ)
     в—ү Why this is useful:
          в—Ӣ The integrand is nonnegative.
          в—Ӣ It is bigger than a simpler function whose improper integral is already known.
     в—ү Check the comparison integral:
          в—Ӣ вҲ«[1, вҲһ] 1/(1 + рқ’ҷ) вӢ… рқ’…рқ’ҷ
          в—Ӣ = lim_(рқ’ғ вҶ’ вҲһ) вҲ«[1, рқ’ғ] 1/(1 + рқ’ҷ) вӢ… рқ’…рқ’ҷ
          в—Ӣ = lim_(рқ’ғ вҶ’ вҲһ) [ln|1 + рқ’ҷ|] | [1, рқ’ғ]
          в—Ӣ = lim_(рқ’ғ вҶ’ вҲһ) [ln(1 + рқ’ғ) вҲ’ ln(2)]
          в—Ӣ = вҲһ
     в—ү Conclusion by comparison:
          в—Ӣ Because
               в–  1/(рқ’ҷ + sinВІ рқ’ҷ) вүҘ 1/(1 + рқ’ҷ)
          в—Ӣ and
               в–  вҲ«[1, вҲһ] 1/(1 + рқ’ҷ) вӢ… рқ’…рқ’ҷ diverges,
          в—Ӣ it follows from the Comparison Test that
               в–  вҲ«[1, вҲһ] 1/(рқ’ҷ + sinВІ рқ’ҷ) вӢ… рқ’…рқ’ҷ also diverges.
     в—ү Final result:
          в—Ӣ The integral is **divergent**.
     в—ү NOTE:
          в—Ӣ Recall the algebra of inequalities for positive quantities:
               в–  If 0 < рқ’Ү(рқ’ҷ) вүӨ рқ’Ҳ(рқ’ҷ), then 1/рқ’Ү(рқ’ҷ) вүҘ 1/рқ’Ҳ(рқ’ҷ).
          в—Ӣ In words:
               в–  a smaller positive denominator gives a larger fraction,
               в–  and a larger positive denominator gives a smaller fraction.
          в—Ӣ This is why, from
               в–  рқ’ҷ + sinВІ(рқ’ҷ) вүӨ 1 + рқ’ҷ,
          в—Ӣ we may conclude that
               в–  1/(рқ’ҷ + sinВІ рқ’ҷ) вүҘ 1/(1 + рқ’ҷ).
