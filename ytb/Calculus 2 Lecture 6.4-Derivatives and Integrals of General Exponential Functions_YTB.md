в—Ҹ [0:05]. Fundamental properties of exponential expressions. [рҹ“·image](./img/Calculus 2 Lecture 6.4/[0-05]Fundamental properties of exponential expressions.png)
     в—ү product of powers with the same base:
          в—Ӣ рқ’Ӯ^рқ’ҷ В· рқ’Ӯ^рқ’ҡ = рқ’Ӯ^{рқ’ҷ+рқ’ҡ}
     в—ү Power of a power:
          в—Ӣ (рқ’Ӯ^рқ’ҷ)^рқ’ҡ = рқ’Ӯ^{рқ’ҷрқ’ҡ}
     в—ү Power of a product:

     в—ү Power of a quotient:

в—Ҹ [1:40]. Rewriting a general exponential function using the natural exponential form. [рҹ“·image](./img/Calculus 2 Lecture 6.4/[1-40]Rewriting a general exponential function using the natural exponential form.png)
     в—ү Expressing any positive base in terms of рқ“®.
          в—Ӣ Use the identity:
               в–  рқ’ҷ = рқ“®^{ln рқ’ҷ}
          в—Ӣ Apply it to a constant base рқ’Ӯ:
               в–  рқ’Ӯ = рқ“®^{ln рқ’Ӯ}
     в—ү Rewriting the exponential function рқ’Ӯ^рқ’ҷ.
          в—Ӣ Substitute рқ’Ӯ = рқ“®^{ln рқ’Ӯ}:
               в–  рқ’Ӯ^рқ’ҷ = (рқ“®^{ln рқ’Ӯ})^рқ’ҷ
          в—Ӣ Use properties of exponents:
               в–  рқ’Ӯ^рқ’ҷ = рқ“®^{рқ’ҷ ln рқ’Ӯ}
     в—ү Representation of a general exponential function.
          в—Ӣ Define the function:
               в–  рқ’Ү(рқ’ҷ) = рқ’Ӯ^рқ’ҷ
          в—Ӣ Equivalent natural exponential form:
               в–  рқ’Ү(рқ’ҷ) = рқ“®^{рқ’ҷ ln рқ’Ӯ}
     в—ү рҹ§© Example вҖ“ with base  2^ПҖ.
          в—Ӣ рқ’Ү(рқ’ҷ) = рқ’Ӯ^рқ’ҷ = рқ“®^{рқ’ҷ ln рқ’Ӯ}
               в–  2^ПҖ = рқ“®^{ПҖ ln 2}

в—Ҹ [6:00]. Derivative of a general exponential function. [рҹ“·image](./img/Calculus 2 Lecture 6.4/[6-00]Derivative of a general exponential function.png)
     в—ү Rewriting the function using the natural exponential.
          в—Ӣ Start from the general exponential:
               в–  рқ’Ү(рқ’ҷ) = рқ’Ӯ^рқ’ҷ
          в—Ӣ Use the identity рқ’Ӯ = рқ“®^{ln рқ’Ӯ}:
               в–  рқ’Ӯ^рқ’ҷ = рқ“®^{рқ’ҷ ln рқ’Ӯ}
     в—ү Apply differentiation rules.
          в—Ӣ Differentiate using the chain rule:
               в–  рқ’…/рқ’…рқ’ҷ [рқ’Ӯ^рқ’ҷ] = рқ’…/рқ’…рқ’ҷ [рқ“®^{рқ’ҷ ln рқ’Ӯ}]
               в–  = рқ“®^{рқ’ҷ ln рқ’Ӯ} В· рқ’…/рқ’…рқ’ҷ [рқ’ҷ ln рқ’Ӯ]
          в—Ӣ Differentiate the exponent:
               в–  рқ’…/рқ’…рқ’ҷ [рқ’ҷ ln рқ’Ӯ] = ln рқ’Ӯ
     в—ү Final result.
          в—Ӣ Substitute back:
               в–  рқ’…/рқ’…рқ’ҷ [рқ’Ӯ^рқ’ҷ] = (ln рқ’Ӯ) рқ“®^{рқ’ҷ ln рқ’Ӯ}
               в–  рқ’…/рқ’…рқ’ҷ [рқ’Ӯ^рқ’ҷ] = (ln рқ’Ӯ) рқ’Ӯ^рқ’ҷ

в—Ҹ [6:00]. рҹ§© Example вҖ“ Differentiate an exponential function with base different from рқ“®: рқ’ҡ=3^рқ’ҷ [рҹ“·image](./img/Calculus 2 Lecture 6.4/[6-00]Example Differentiate an exponential function with base different from рқ“® 3^рқ’ҷ.png)
     в—ү Apply the general derivative formula.
          в—Ӣ рқ’…/рқ’…рқ’ҷ [рқ’Ӯ^рқ’ҷ] = (ln рқ’Ӯ) рқ’Ӯ^рқ’ҷ
     в—ү Example with base 3.
          в—Ӣ рқ’…/рқ’…рқ’ҷ [3^рқ’ҷ] = (ln 3) В· 3^рқ’ҷ

в—Ҹ [12:52]. рҹ§© Example вҖ“ Differentiate an exponential function with base different from рқ“®: рқ’ҡ=5^{вҲӣрқ’ҷ} [рҹ“·image](./img/Calculus 2 Lecture 6.4/[12-52]Example Differentiate an exponential function with base different from рқ“® 5^{вҲӣрқ’ҷ}.png)
     в—ү Apply the derivative rule for рқ’Ӯ^{рқ’–}:
          в—Ӣ рқ’…/рқ’…рқ’ҷ [рқ’Ӯ^{рқ’–}] = (ln рқ’Ӯ) рқ’Ӯ^{рқ’–} В· рқ’…рқ’–/рқ’…рқ’ҷ
     в—ү Identify the inner function:
          в—Ӣ рқ’– = вҲӣрқ’ҷ = рқ’ҷ^{1/3}
     в—ү Differentiate the inner function:
          в—Ӣ рқ’…/рқ’…рқ’ҷ [рқ’ҷ^{1/3}] = (1/3) рқ’ҷ^{-2/3}
     в—ү Combine all factors:
          в—Ӣ рқ’…/рқ’…рқ’ҷ [5^{вҲӣрқ’ҷ}] = (ln 5) В· 5^{вҲӣрқ’ҷ} В· (1/3) рқ’ҷ^{-2/3}
     в—ү Final simplified form:
          в—Ӣ рқ’…/рқ’…рқ’ҷ [5^{вҲӣрқ’ҷ}] = (ln 5) В· 5^{вҲӣрқ’ҷ} / (3вҲӣрқ’ҷВІ)

в—Ҹ [16:45]. рҹ§© Example вҖ“ Differentiate an exponential function with base different from рқ“®: рқ’ҡ = 7^{tan(2рқ’ҷ)} [рҹ“·image](./img/Calculus 2 Lecture 6.4/[16-45]Example Differentiate an exponential function with base different from рқ“® рқ’ҡ = 7^{tan(2рқ’ҷ)}.png)
     в—ү Given function:
          в—Ӣ рқ’ҡ = 7^{tan(2рқ’ҷ)}
     в—ү Apply the derivative rule for рқ’Ӯ^{рқ’–}:
          в—Ӣ рқ’…/рқ’…рқ’ҷ [рқ’Ӯ^{рқ’–}] = (ln рқ’Ӯ) рқ’Ӯ^{рқ’–} В· рқ’…рқ’–/рқ’…рқ’ҷ
     в—ү Identify the inner function:
          в—Ӣ рқ’– = tan(2рқ’ҷ)
     в—ү Differentiate the inner function (chain rule):
          в—Ӣ рқ’…/рқ’…рқ’ҷ [tan(2рқ’ҷ)] = secВІ(2рқ’ҷ) В· рқ’…/рқ’…рқ’ҷ[2рқ’ҷ]
          в—Ӣ рқ’…/рқ’…рқ’ҷ [tan(2рқ’ҷ)] = 2 secВІ(2рқ’ҷ)
     в—ү Combine all factors:
          в—Ӣ рқ’…/рқ’…рқ’ҷ [7^{tan(2рқ’ҷ)}] = (ln 7) В· 7^{tan(2рқ’ҷ)} В· 2 secВІ(2рқ’ҷ)
     в—ү Final result:
          в—Ӣ рқ’…/рқ’…рқ’ҷ [7^{tan(2рқ’ҷ)}] = 2 (ln 7) В· 7^{tan(2рқ’ҷ)} В· secВІ(2рқ’ҷ)

в—Ҹ [20:35]. рҹ§© Example вҖ“ Differentiate a product involving an exponential function:  рқ’ҡ = рқ’ҷ^рқ“® В· рқ“®^рқ’ҷ [рҹ“·image](./img/Calculus 2 Lecture 6.4/[20-35]Example Differentiate a product involving an exponential function рқ’ҡ = рқ’ҷ^рқ“® В· рқ“®^рқ’ҷ.png)
     в—ү Apply the product rule:
          в—Ӣ рқ’…/рқ’…рқ’ҷ[рқ’–В·рқ’—] = рқ’–'В·рқ’— + рқ’–В·рқ’—'
     в—ү Identify each factor:
          в—Ӣ рқ’– = рқ’ҷ^рқ“®
          в—Ӣ рқ’— = рқ“®^рқ’ҷ
     в—ү Differentiate each factor:
          в—Ӣ рқ’…/рқ’…рқ’ҷ[рқ’ҷ^рқ“®] = рқ“®В·рқ’ҷ^{рқ“®вҲ’1}
          в—Ӣ рқ’…/рқ’…рқ’ҷ[рқ“®^рқ’ҷ] = рқ“®^рқ’ҷ
     в—ү Substitute into the product rule:
          в—Ӣ рқ’…рқ’ҡ/рқ’…рқ’ҷ = (рқ“®В·рқ’ҷ^{рқ“®вҲ’1})В·рқ“®^рқ’ҷ + рқ’ҷ^рқ“®В·рқ“®^рқ’ҷ
     в—ү Factor the common exponential term:
          в—Ӣ рқ’…рқ’ҡ/рқ’…рқ’ҷ = рқ“®^рқ’ҷ (рқ“®В·рқ’ҷ^{рқ“®вҲ’1} + рқ’ҷ^рқ“®)

в—Ҹ [24:13]. рҹ§© Example вҖ“ Differentiate a quotient involving exponential functions with different bases: рқ’ҡ = 2^рқ’ҷ / вҲҡ(3^рқ’ҷ + 1) [рҹ“·image](./img/Calculus 2 Lecture 6.4/[24-13]Example Differentiate a quotient involving exponential functions with different bases рқ’ҡ = 2^рқ’ҷ вҲ• вҲҡ(3^рқ’ҷ + 1).png)
     в—ү Rewrite the denominator using exponents:
          в—Ӣ вҲҡ(3^рқ’ҷ + 1) = (3^рқ’ҷ + 1)^{1/2}
     в—ү Rewrite the function in exponent form:
          в—Ӣ рқ’ҡ = 2^рқ’ҷ В· (3^рқ’ҷ + 1)^{-1/2}
     в—ү Apply the product rule:
          в—Ӣ рқ’…/рқ’…рқ’ҷ[рқ’–В·рқ’—] = рқ’–'В·рқ’— + рқ’–В·рқ’—'
     в—ү Identify each factor:
          в—Ӣ рқ’– = 2^рқ’ҷ
          в—Ӣ рқ’— = (3^рқ’ҷ + 1)^{-1/2}
     в—ү Differentiate each factor:
          в—Ӣ рқ’…/рқ’…рқ’ҷ[2^рқ’ҷ] = (ln 2)В·2^рқ’ҷ
          в—Ӣ рқ’…/рқ’…рқ’ҷ[(3^рқ’ҷ + 1)^{-1/2}]
               в–  = вҲ’1/2 (3^рқ’ҷ + 1)^{-3/2} В· рқ’…/рқ’…рқ’ҷ[3^рқ’ҷ]
               в–  = вҲ’1/2 (3^рқ’ҷ + 1)^{-3/2} В· (ln 3)В·3^рқ’ҷ
     в—ү Substitute into the product rule:
          в—Ӣ рқ’…рқ’ҡ/рқ’…рқ’ҷ =
               в–  (ln 2)В·2^рқ’ҷ (3^рқ’ҷ + 1)^{-1/2}
               в–  вҲ’ 2^рқ’ҷ В· (1/2)(ln 3)В·3^рқ’ҷ (3^рқ’ҷ + 1)^{-3/2}
     в—ү Factor common terms:
          в—Ӣ рқ’…рқ’ҡ/рқ’…рқ’ҷ = 2^рқ’ҷ (3^рқ’ҷ + 1)^{-3/2}
               в–  [ (ln 2)(3^рқ’ҷ + 1) вҲ’ (1/2)(ln 3)В·3^рқ’ҷ ]


в—Ҹ [24:53]. рҹ§© Example вҖ“ Differentiate a product involving a general exponential function:  рқ’ҡ = рқ’ҷ В· 5^{3рқ’ҷ} [рҹ“·image](./img/Calculus 2 Lecture 6.4/[24-53]Example Differentiate a product involving a general exponential function  рқ’ҡ = рқ’ҷ В· 5^{3рқ’ҷ}.png)
     в—ү Apply the product rule:
          в—Ӣ рқ’…/рқ’…рқ’ҷ[рқ’–В·рқ’—] = рқ’–'В·рқ’— + рқ’–В·рқ’—'
     в—ү Identify each factor:
          в—Ӣ рқ’– = рқ’ҷ

     в—ү Differentiate each factor:
          в—Ӣ рқ’…/рқ’…рқ’ҷ[рқ’ҷ] = 1

     в—ү Simplify the expression:

     в—ү Factor the common exponential term:

в—Ҹ [28:25]. рҹ§© Example вҖ“ Logarithmic differentiation for a variable base and exponent: рқ’ҡ = рқ’ҷ^рқ’ҷ [рҹ“·image](./img/Calculus 2 Lecture 6.4/[28-25]Example Logarithmic differentiation for a variable base and exponent рқ’ҡ = рқ’ҷ^рқ’ҷ.png)
     в—ү **Logarithmic differentiation for a variable base and exponent:** [рҹ“·image](./img/Calculus 2 Lecture 6.4/[28-25]Logarithmic differentiation for a variable base and exponent.png)
          в—Ӣ Use this technique when the function has the form рқ’ҡ = (рқ’Ү(рқ’ҷ))^{рқ’Ҳ(рқ’ҷ)}, where both the base and the exponent depend on рқ’ҷ.
          в—Ӣ Idea: take natural logarithms to simplify exponents into products, which are easier to differentiate.
               в–  ln(рқ’ҡ) = ln((рқ’Ү(рқ’ҷ))^{рқ’Ҳ(рқ’ҷ)}) = рқ’Ҳ(рқ’ҷ) В· ln(рқ’Ү(рқ’ҷ))
          в—Ӣ Differentiate both sides with respect to рқ’ҷ using implicit differentiation.
          в—Ӣ This method avoids complicated applications of the chain and product rules directly on the original expression.
          в—Ӣ Especially useful for expressions like рқ’ҷ^рқ’ҷ, (sin рқ’ҷ)^{рқ’ҷ}, or (рқ’ҷВІ+1)^{вҲҡрқ’ҷ}.
     в—ү Take natural logarithms on both sides:
          в—Ӣ ln(рқ’ҡ) = ln(рқ’ҷ^рқ’ҷ)
     в—ү Use the power rule for logarithms:
          в—Ӣ ln(рқ’ҡ) = рқ’ҷ ln(рқ’ҷ)
     в—ү Differentiate both sides with respect to рқ’ҷ:
          в—Ӣ рқ’…/рқ’…рқ’ҷ[ln(рқ’ҡ)] = рқ’…/рқ’…рқ’ҷ[рқ’ҷ ln(рқ’ҷ)]
     в—ү Apply implicit differentiation on the left:
          в—Ӣ (1/рқ’ҡ) В· рқ’…рқ’ҡ/рқ’…рқ’ҷ = рқ’…/рқ’…рқ’ҷ[рқ’ҷ]В·ln(рқ’ҷ) + рқ’ҷВ·рқ’…/рқ’…рқ’ҷ[ln(рқ’ҷ)]
     в—ү Compute each derivative:
          в—Ӣ (1/рқ’ҡ) В· рқ’…рқ’ҡ/рқ’…рқ’ҷ = ln(рқ’ҷ) + рқ’ҷВ·(1/рқ’ҷ)
     в—ү Simplify:
          в—Ӣ (1/рқ’ҡ) В· рқ’…рқ’ҡ/рқ’…рқ’ҷ = ln(рқ’ҷ) + 1
     в—ү Multiply both sides by рқ’ҡ:
          в—Ӣ рқ’…рқ’ҡ/рқ’…рқ’ҷ = (ln(рқ’ҷ) + 1) В· рқ’ҡ
     в—ү Substitute back рқ’ҡ = рқ’ҷ^рқ’ҷ:
          в—Ӣ рқ’…рқ’ҡ/рқ’…рқ’ҷ = (ln(рқ’ҷ) + 1) В· рқ’ҷ^рқ’ҷ


в—Ҹ [35:40]. рҹ§© Example вҖ“ Logarithmic differentiation for a variable base and exponent: рқ’ҡ = (рқ’ҷВІ + рқ’ҷ)^{вҲҡрқ’ҷ}. [рҹ“·image](./img/Calculus 2 Lecture 6.4/[35-40]Example Logarithmic differentiation for a variable base and exponent рқ’ҡ = (рқ’ҷВІ + рқ’ҷ)^{вҲҡрқ’ҷ}.png)
     в—ү Take natural logarithms:
          в—Ӣ ln(рқ’ҡ) = ln((рқ’ҷВІ + рқ’ҷ)^{вҲҡрқ’ҷ})
          в—Ӣ ln(рқ’ҡ) = вҲҡрқ’ҷ В· ln(рқ’ҷВІ + рқ’ҷ)
     в—ү Differentiate both sides with respect to рқ’ҷ:
          в—Ӣ рқ’…/рқ’…рқ’ҷ[ln(рқ’ҡ)] = рқ’…/рқ’…рқ’ҷ[вҲҡрқ’ҷ В· ln(рқ’ҷВІ + рқ’ҷ)]
     в—ү Implicit differentiation on the left:
          в—Ӣ (1/рқ’ҡ) В· рқ’…рқ’ҡ/рқ’…рқ’ҷ = рқ’…/рқ’…рқ’ҷ[вҲҡрқ’ҷ] В· ln(рқ’ҷВІ + рқ’ҷ) + вҲҡрқ’ҷ В· рқ’…/рқ’…рқ’ҷ[ln(рқ’ҷВІ + рқ’ҷ)]
     в—ү Compute each derivative:
          в—Ӣ рқ’…/рқ’…рқ’ҷ[вҲҡрқ’ҷ] = 1/(2вҲҡрқ’ҷ)
          в—Ӣ рқ’…/рқ’…рқ’ҷ[ln(рқ’ҷВІ + рқ’ҷ)] = (1/(рқ’ҷВІ + рқ’ҷ)) В· рқ’…/рқ’…рқ’ҷ[рқ’ҷВІ + рқ’ҷ]
          в—Ӣ рқ’…/рқ’…рқ’ҷ[рқ’ҷВІ + рқ’ҷ] = 2рқ’ҷ + 1
     в—ү Substitute and simplify:
          в—Ӣ (1/рқ’ҡ) В· рқ’…рқ’ҡ/рқ’…рқ’ҷ = ln(рқ’ҷВІ + рқ’ҷ)/(2вҲҡрқ’ҷ) + вҲҡрқ’ҷ(2рқ’ҷ + 1)/(рқ’ҷВІ + рқ’ҷ)
     в—ү Solve for рқ’…рқ’ҡ/рқ’…рқ’ҷ:
          в—Ӣ рқ’…рқ’ҡ/рқ’…рқ’ҷ = [ ln(рқ’ҷВІ + рқ’ҷ)/(2вҲҡрқ’ҷ) + вҲҡрқ’ҷ(2рқ’ҷ + 1)/(рқ’ҷВІ + рқ’ҷ) ] В· рқ’ҡ
     в—ү Substitute back рқ’ҡ = (рқ’ҷВІ + рқ’ҷ)^{вҲҡрқ’ҷ}:
          в—Ӣ рқ’…рқ’ҡ/рқ’…рқ’ҷ = [ ln(рқ’ҷВІ + рқ’ҷ)/(2вҲҡрқ’ҷ) + вҲҡрқ’ҷ(2рқ’ҷ + 1)/(рқ’ҷВІ + рқ’ҷ) ] В· (рқ’ҷВІ + рқ’ҷ)^{вҲҡрқ’ҷ}

в—Ҹ [1:00:24]. рҹ§© Example вҖ“ Differentiation using the chain rule (not logarithmic differentiation).
     в—ү Given function with constant irrational exponent:
          в—Ӣ рқ’ҡ = (рқ’ҷ + cos рқ’ҷ)^{вҲҡ2}
     в—ү Apply the power rule combined with the chain rule:
          в—Ӣ Outer function: рқ’–^{вҲҡ2}
          в—Ӣ Inner function: рқ’– = рқ’ҷ + cos рқ’ҷ
     в—ү Differentiate step by step:
          в—Ӣ рқ’…/рқ’…рқ’ҷ [рқ’ҡ] = вҲҡ2 В· (рқ’ҷ + cos рқ’ҷ)^{вҲҡ2 вҲ’ 1} В· рқ’…/рқ’…рқ’ҷ(рқ’ҷ + cos рқ’ҷ)

     в—ү Final derivative:

в—Ҹ [46:50]. рҹ§© Example вҖ“ Definite integral of an exponential function with base different from рқ“®:  вҲ«[0,4] 3^рқ’ҷ рқ’…рқ’ҷ [рҹ“·image](./img/Calculus 2 Lecture 6.4/[46-50]Example Definite integral of an exponential function with base different from рқ“®  вҲ«[0,4] 3^рқ’ҷ рқ’…рқ’ҷ.png)

в—Ҹ [49:45]. рҹ§© Example вҖ“ Definite integral using substitution: вҲ«[1,4] 3^{вҲҡрқ’ҷ} / вҲҡрқ’ҷ  рқ’…рқ’ҷ [рҹ“·image](./img/Calculus 2 Lecture 6.4/[49-45]Example Definite integral using substitution вҲ«[1,4] 3^{вҲҡрқ’ҷ} вҲ• вҲҡрқ’ҷ  рқ’…рқ’ҷ.png)

в—Ҹ [57:20]. рҹ§© Examples вҖ“ рқ’–-substitution. [рҹ“·image](./img/Calculus 2 Lecture 6.4/[57-20]Examples рқ’–-substitution.png)
