     в—ү The domain is:
          в—Ӣ the set of рқ’ҷ-values for which the series converges.
     в—ү Important:
          в—Ӣ not every рқ’ҷ-value necessarily works.
     в—ү For some values of рқ’ҷ:

     в—ү For other values of рқ’ҷ:

в—Ҹ [16:47](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=1007). Input and output.
     в—ү Input:
          в—Ӣ a value of рқ’ҷ.
     в—ү After plugging in рқ’ҷ:
          в—Ӣ the power series becomes an ordinary numerical series.
     в—ү If that numerical series converges:
          в—Ӣ its sum becomes the output рқ’Ү(рқ’ҷ).
               в–  this only happens for values of рқ’ҷ where the series converges.
     в—ү If that numerical series diverges:
          в—Ӣ there is no valid output for the function at that рқ’ҷ.

в—Ҹ NМІOМІTМІEМІ вҖ” Key interpretation.
     в—ү рқ’ҷ is the input.
     в—ү The sum of the convergent series is the output.
     в—ү So:
          в—Ӣ рқ’Ү(рқ’ҷ) = sum of the power series.
     в—ү But this only works:
          в—Ӣ for рқ’ҷ-values inside the interval of convergence.

     в—ү The distance from рқ‘Ә to the endpoints is called:

     в—ү Therefore:
          в—Ӣ the center рқ‘Ә sits in the middle of the interval of convergence.

в—Ҹ [36:33](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=2193). Three possible convergence cases.
     в—ү Case A:
          в—Ӣ convergence only at the center.
     в—ү Case B:
          в—Ӣ convergence for all real рқ’ҷ.
     в—ү Case C:
          в—Ӣ convergence inside a finite interval around the center.

в—Ҹ [37:00](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=2220). Case A вҖ” Convergence only at the center.
     в—ү The series converges only when:
          в—Ӣ рқ’ҷ = рқ‘Ә
     в—ү Then:
          в—Ӣ radius of convergence рқ‘№ = 0
     в—ү Meaning:
          в—Ӣ there is no interval around рқ‘Ә.
          в—Ӣ the only valid point is the center itself.
     в—ү Symbolically:
          в—Ӣ 0 вүӨ |рқ’ҷ - рқ‘Ә| вүӨ 0
     в—ү Therefore:
          в—Ӣ рқ’ҷ = рқ‘Ә only.

в—Ҹ [38:37](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=2317). Case B вҖ” Convergence for all real рқ’ҷ.
     в—ү The series converges for:
          в—Ӣ every real value of рқ’ҷ.
     в—ү Then:
          в—Ӣ radius of convergence рқ‘№ = вҲһ
     в—ү Symbolically:
          в—Ӣ -вҲһ < рқ’ҷ - рқ‘Ә < вҲһ
     в—ү Interval of convergence:
          в—Ӣ (-вҲһ,вҲһ)

в—Ҹ [39:35](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=2375). Case C вҖ” Finite radius of convergence.
     в—ү The series converges when:
          в—Ӣ |рқ’ҷ - рқ‘Ә| < рқ‘№
     в—ү It diverges when:
          в—Ӣ |рқ’ҷ - рқ‘Ә| > рқ‘№
     в—ү Here:
          в—Ӣ рқ‘№ is the radius of convergence.
     в—ү Meaning:
          в—Ӣ рқ‘№ tells how far we can move left and right from the center рқ‘Ә while the series still converges.
     в—ү Interval from the radius.
          в—Ӣ Start with:
               в–  |рқ’ҷ - рқ‘Ә| < рқ‘№
          в—Ӣ Equivalent inequality:
               в–  -рқ‘№ < рқ’ҷ - рқ‘Ә < рқ‘№
          в—Ӣ Add рқ‘Ә:
               в–  рқ‘Ә - рқ‘№ < рқ’ҷ < рқ‘Ә + рқ‘№
          в—Ӣ Therefore:
               в–  the interval is centered at рқ‘Ә.
               в–  the radius is рқ‘№.
               в–  the preliminary interval is (рқ‘Ә - рқ‘№, рқ‘Ә + рқ‘№).
     в—ү Endpoints must be checked separately.
          в—Ӣ The Ratio Test usually gives:
               в–  |рқ’ҷ - рқ‘Ә| < рқ‘№
          в—Ӣ But it does not decide:
               в–  рқ’ҷ = рқ‘Ә - рқ‘№
               в–  рқ’ҷ = рқ‘Ә + рқ‘№
          в—Ӣ Therefore:
               в–  plug each endpoint into the original power series.
               в–  test the resulting numerical series separately.
          в—Ӣ Use any appropriate convergence test:
               в–  Alternating Series Test.
               в–  рқ’‘-Series Test.
               в–  Comparison Test.
               в–  Limit Comparison Test.
               в–  etc.

     в—ү This is a power series centered at:
          в—Ӣ 0
     в—ү The factorial suggests:
          в—Ӣ use the Ratio Test.
     в—ү Goal:
          в—Ӣ determine for which рқ’ҷ-values the power series converges.

в—Ҹ [50:14](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=3014). Use the Ratio Test.
     в—ү Let:
          в—Ӣ рқ’ӮвӮҷ = рқ’Ҹ!рқ’ҷвҒҝ
     в—ү Then:
          в—Ӣ рқ’ӮвӮҷвӮҠвӮҒ = (рқ’Ҹ + 1)!рқ’ҷвҒҝвҒәВ№
     в—ү Compute:
          в—Ӣ рқ‘і = limвӮҷвҶ’вҲһ |рқ’ӮвӮҷвӮҠвӮҒ/рқ’ӮвӮҷ|
          в—Ӣ = limвӮҷвҶ’вҲһ |[(рқ’Ҹ + 1)!рқ’ҷвҒҝвҒәВ№]/[рқ’Ҹ!рқ’ҷвҒҝ]|
     в—ү Simplify:
          в—Ӣ (рқ’Ҹ + 1)!/рқ’Ҹ! = рқ’Ҹ + 1
          в—Ӣ рқ’ҷвҒҝвҒәВ№/рқ’ҷвҒҝ = рқ’ҷ
     в—ү Therefore:
          в—Ӣ рқ‘і = limвӮҷвҶ’вҲһ (рқ’Ҹ + 1)|рқ’ҷ|

в—Ҹ [51:47](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=3107). Interpreting the Ratio Test result.
     в—ү If рқ’ҷ вү  0:
          в—Ӣ |рқ’ҷ| is a positive constant.
          в—Ӣ limвӮҷвҶ’вҲһ (рқ’Ҹ + 1)|рқ’ҷ| = вҲһ
          в—Ӣ вҲһ > 1
          в—Ӣ the series diverges by the Ratio Test.
     в—ү If рқ’ҷ = 0:
          в—Ӣ limвӮҷвҶ’вҲһ (рқ’Ҹ + 1)|0| = 0
          в—Ӣ 0 < 1
          в—Ӣ the series converges by the Ratio Test.
     в—ү Therefore:
          в—Ӣ the series converges only at рқ’ҷ = 0.

в—Ҹ [53:25](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=3205). Radius and domain.
     в—ү Center:
          в—Ӣ рқ‘Ә = 0
     в—ү Domain:
          в—Ӣ рқ’ҷ = 0
     в—ү Radius of convergence:
          в—Ӣ рқ‘№ = 0
     в—ү Interpretation:
          в—Ӣ the power series represents a function only at the center.

в—Ҹ NМІOМІTМІEМІ вҖ” Key idea.
     в—ү The factorial рқ’Ҹ! grows so fast that any рқ’ҷ вү  0 makes the series diverge.
     в—ү Only the center survives:
          в—Ӣ рқ’ҷ = 0
     в—ү Therefore:
          в—Ӣ the radius of convergence is рқ‘№ = 0.

     в—ү This is a power series centered at:

     в—ү Since factorials appear:

     в—ү Goal:

     в—ү Then:

     в—ү Compute:

в—Ҹ [1:00:00](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=3600). Build the ratio.
     в—ү Substitute:
          в—Ӣ рқ‘і = limвӮҷвҶ’вҲһ | [(-1)вҒҝвҒәВ№рқ’ҷВІвҒҝвҒәВІ/(2рқ’Ҹ + 2)!] / [(-1)вҒҝрқ’ҷВІвҒҝ/(2рқ’Ҹ)!] |
     в—ү Reciprocate and multiply:

в—Ҹ [1:01:00](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=3660). Simplify signs and powers.
     в—ү The absolute value removes the alternating sign:
          в—Ӣ |(-1)вҒҝвҒәВ№/(-1)вҒҝ| = 1
     в—ү Simplify the powers of рқ’ҷ:
          в—Ӣ рқ’ҷВІвҒҝвҒәВІ/рқ’ҷВІвҒҝ = рқ’ҷВІ
     в—ү Therefore:

в—Ҹ [1:02:35](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=3755). Factorial simplification.
     в—ү Use:

     в—ү Therefore:
          в—Ӣ (2рқ’Ҹ)!/(2рқ’Ҹ + 2)! = 1/[(2рқ’Ҹ + 2)(2рқ’Ҹ + 1)]
     в—ү So:
          в—Ӣ рқ‘і = limвӮҷвҶ’вҲһ рқ’ҷВІ/[(2рқ’Ҹ + 2)(2рқ’Ҹ + 1)]

в—Ҹ [1:06:44](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=4004). Limit result.
     в—ү For any fixed рқ’ҷ:
          в—Ӣ рқ’ҷВІ is a constant.
          в—Ӣ (2рқ’Ҹ + 2)(2рқ’Ҹ + 1) вҶ’ вҲһ.
     в—ү Therefore:
          в—Ӣ рқ‘і = limвӮҷвҶ’вҲһ рқ’ҷВІ/[(2рқ’Ҹ + 2)(2рқ’Ҹ + 1)] = 0
     в—ү This happens for all real рқ’ҷ.

в—Ҹ [1:08:27](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=4107). Conclusion for Example 2.
     в—ү Since:
          в—Ӣ рқ‘і = 0 < 1
     в—ү By the Ratio Test:
          в—Ӣ the power series converges for all рқ’ҷ.
     в—ү Therefore:
          в—Ӣ radius of convergence рқ‘№ = вҲһ
          в—Ӣ interval of convergence is (-вҲһ,вҲһ)

в—Ҹ NМІOМІTМІEМІ 1 вҖ” Key idea.
     в—ү The factorial in the denominator grows fast enough to dominate every fixed power of рқ’ҷ.
     в—ү No matter what real value рқ’ҷ has:
          в—Ӣ the ratio limit becomes 0.
     в—ү Therefore:
          в—Ӣ the series converges for all real рқ’ҷ.
          
в—Ҹ NМІOМІTМІEМІ 2 вҖ” Why the domain is all real numbers.
     в—ү After applying the Ratio Test, we get:
          в—Ӣ рқ‘і = limвӮҷвҶ’вҲһ рқ’ҷВІ/[(2рқ’Ҹ + 2)(2рқ’Ҹ + 1)]
     в—ү Important:
          в—Ӣ рқ’ҷ is fixed.
          в—Ӣ рқ’Ҹ is the variable that goes to infinity.
     в—ү For any fixed real value of рқ’ҷ:
          в—Ӣ рқ’ҷВІ is just a constant.
          в—Ӣ (2рқ’Ҹ + 2)(2рқ’Ҹ + 1) вҶ’ вҲһ.
     в—ү Therefore:
          в—Ӣ constant / вҲһ = 0
          в—Ӣ рқ‘і = 0
     в—ү Since:
          в—Ӣ 0 < 1
     в—ү By the Ratio Test:
          в—Ӣ the series converges for every real value of рқ’ҷ.
     в—ү Therefore:
          в—Ӣ domain = (-вҲһ,вҲһ)
          в—Ӣ radius of convergence рқ‘№ = вҲһ
     в—ү Key idea:
          в—Ӣ we do not plug infinity into рқ’ҷ.
          в—Ӣ рқ’ҷ stays fixed.
          в—Ӣ only рқ’Ҹ goes to infinity.

пј–.2 - пј©пҪҺпҪ”пҪ…пҪҮпҪ’пҪҒпҪҢпҪ“гҖҖпҪҸпҪҶгҖҖпј°пҪҸпҪ—пҪ…пҪ’гҖҖпјіпҪ…пҪ’пҪүпҪ…пҪ“

в—Ҹ [2:12:35](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=7955). Integral term by term.
     в—ү We can integrate a power series term by term.
     в—ү Starting from:
          в—Ӣ рқ’Ү(рқ’ҷ) = вҲ‘[рқ’Ҹ=0,вҲһ] рқ’ӮвӮҷ(рқ’ҷ - рқ‘Ә)вҒҝ
     в—ү Each term integrates using:
          в—Ӣ power rule for integrals.

в—Ҹ [2:14:50](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=8090). General integral formula.
     в—ү The integral is:
          в—Ӣ вҲ« рқ’Ү(рқ’ҷ) вӢ… рқ’…рқ‘Ҙ = рқ‘І + вҲ‘[рқ’Ҹ=0,вҲһ] рқ’ӮвӮҷ(рқ’ҷ - рқ‘Ә)вҒҝвҒәВ№/(рқ’Ҹ + 1)
     в—ү Important:
          в—Ӣ include the constant of integration.
     в—ү The series still starts at:
          в—Ӣ рқ’Ҹ = 0

в—Ҹ NМІOМІTМІEМІ вҖ” Endpoints may change.
     в—ү Differentiating or integrating a power series:
          в—Ӣ keeps the same radius of convergence.
     в—ү But:
          в—Ӣ endpoint behavior may change.
     в—ү A derivative can lose endpoints.
     в—ү An integral can gain endpoints.
     в—ү Therefore:
          в—Ӣ endpoints must be checked separately when needed.




пј–. пј“  -  пјЎпҪҗпҪҗпҪҢпҪүпҪғпҪҒпҪ”пҪүпҪҸпҪҺ: пј°пҪҸпҪ—пҪ…пҪ’гҖҖпјіпҪ…пҪ’пҪүпҪ…пҪ“гҖҖпјҰпҪҸпҪ’гҖҖпј¬пҪҺпјҲпј‘пјҚрқ’ҷпјү

в—Ҹ [рҹ“·image-1](../img/Calculus 2 Lecture 9.7/[2-17-26]-01.png)  [рҹ“·image-2](../img/Calculus 2 Lecture 9.7/[2-17-26]-02.png)

в—Ҹ [2:17:26](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=8246). рҹ§© Application вҖ” Find a power series representation for Ln(1 - рқ’ҷ).
     в—ү Goal:
          в—Ӣ find a power series representation for:
               в–  Ln(1 - рқ’ҷ)
     в—ү Interval:
          в—Ӣ -1 < рқ’ҷ < 1
     в—ү Main idea:
          в—Ӣ start from a known power series,
          в—Ӣ then integrate term by term to create the logarithm.

в—Ҹ [2:17:49](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=8269). Choosing the starting function.
     в—ү We want:
          в—Ӣ Ln(1 - рқ’ҷ)
     в—ү LeonardвҖҷs strategy:
          в—Ӣ look for a simpler function whose integral gives Ln(1 - рқ’ҷ).
     в—ү Since:
          в—Ӣ the integral of 1/(1 - рқ’ҷ) is very close to Ln(1 - рқ’ҷ),
     в—ү We start with:
          в—Ӣ 1/(1 - рқ’ҷ)
     в—ү Important:
          в—Ӣ вҲ« 1/(1 - рқ’ҷ) вӢ… рқ’…рқ’ҷ = -Ln(1 - рқ’ҷ)
     в—ү The negative sign appears because:
          в—Ӣ the derivative of 1 - рқ’ҷ is -1.

в—Ҹ [2:18:36](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=8316). Why 1/(1 - рқ’ҷ) is useful.
     в—ү The function:
          в—Ӣ 1/(1 - рқ’ҷ)
     в—ү is the sum of a geometric power series.
     в—ү Geometric series formula:
          в—Ӣ 1/(1 - рқ’“) = 1 + рқ’“ + рқ’“ВІ + рқ’“Ві + ...
     в—ү In this case:
          в—Ӣ рқ’“ = рқ’ҷ
     в—ү Therefore:
          в—Ӣ 1/(1 - рқ’ҷ) = 1 + рқ’ҷ + рқ’ҷВІ + рқ’ҷВі + ...
     в—ү Series notation:
          в—Ӣ 1/(1 - рқ’ҷ) = вҲ‘[рқ’Ҹ=0,вҲһ] рқ’ҷвҒҝ
          в—Ӣ or equivalently:
          в—Ӣ 1/(1 - рқ’ҷ) = вҲ‘[рқ’Ҹ=1,вҲһ] рқ’ҷвҒҝвҒ»В№

в—Ҹ [2:21:58](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=8518). Why the interval is (-1,1).
     в—ү This interval comes from the geometric series.
     в—ү A geometric series converges when:
          в—Ӣ |рқ’“| < 1
     в—ү Here:
          в—Ӣ рқ’“ = рқ’ҷ
     в—ү Therefore:
          в—Ӣ |рқ’ҷ| < 1
     в—ү Equivalent:
          в—Ӣ -1 < рқ’ҷ < 1
     в—ү Meaning:
          в—Ӣ the power series for 1/(1 - рқ’ҷ) is valid only on (-1,1).
     в—ү Since we are using that power series as our starting point,
          в—Ӣ the representation for Ln(1 - рқ’ҷ) is also built on this interval.

в—Ҹ [2:24:49](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=8689). Integrate both sides.
     в—ү Start with:
          в—Ӣ 1/(1 - рқ’ҷ) = 1 + рқ’ҷ + рқ’ҷВІ + рқ’ҷВі + ...
     в—ү Integrate both sides:
          в—Ӣ вҲ« 1/(1 - рқ’ҷ) вӢ… рқ’…рқ’ҷ = вҲ«(1 + рқ’ҷ + рқ’ҷВІ + рқ’ҷВі + ...) вӢ… рқ’…рқ’ҷ
     в—ү Left side:
          в—Ӣ вҲ« 1/(1 - рқ’ҷ) вӢ… рқ’…рқ’ҷ = -Ln(1 - рқ’ҷ)
     в—ү Right side:
          в—Ӣ вҲ«(1 + рқ’ҷ + рқ’ҷВІ + рқ’ҷВі + ...) вӢ… рқ’…рқ’ҷ
          в—Ӣ = рқ’ҷ + рқ’ҷВІ/2 + рқ’ҷВі/3 + рқ’ҷвҒҙ/4 + ... + рқ‘І

в—Ҹ [2:26:17](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=8777). Solve for Ln(1 - рқ’ҷ).
     в—ү We have:
          в—Ӣ -Ln(1 - рқ’ҷ) = рқ’ҷ + рқ’ҷВІ/2 + рқ’ҷВі/3 + рқ’ҷвҒҙ/4 + ... + рқ‘І
     в—ү Multiply by -1:
          в—Ӣ Ln(1 - рқ’ҷ) = -рқ’ҷ - рқ’ҷВІ/2 - рқ’ҷВі/3 - рқ’ҷвҒҙ/4 - ... - рқ‘І

в—Ҹ [2:27:02](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=8822). Determine the constant.
     в—ү Plug in:
          в—Ӣ рқ’ҷ = 0
     в—ү Left side:
          в—Ӣ Ln(1 - 0) = Ln(1) = 0
     в—ү Right side:
          в—Ӣ all terms with рқ’ҷ become 0.
     в—ү Therefore:
          в—Ӣ 0 = -рқ‘І
     в—ү So:
          в—Ӣ рқ‘І = 0

в—Ҹ [2:27:31](https://www.youtube.com/watch?v=TGD-TP1c7i4&t=8851). Final power series representation.
     в—ү Therefore:
          в—Ӣ Ln(1 - рқ’ҷ) = -рқ’ҷ - рқ’ҷВІ/2 - рқ’ҷВі/3 - рқ’ҷвҒҙ/4 - ...
     в—ү Factor the negative sign:
          в—Ӣ Ln(1 - рқ’ҷ) = -(рқ’ҷ + рқ’ҷВІ/2 + рқ’ҷВі/3 + рқ’ҷвҒҙ/4 + ...)
     в—ү Series notation:
          в—Ӣ Ln(1 - рқ’ҷ) = -вҲ‘[рқ’Ҹ=1,вҲһ] рқ’ҷвҒҝ/рқ’Ҹ
     в—ү Valid on:
          в—Ӣ (-1,1)

в—Ҹ NМІOМІTМІEМІ 1 вҖ” Why this method works.
     в—ү We do not invent the logarithm series from nothing.
     в—ү We start from the geometric power series:
          в—Ӣ 1/(1 - рқ’ҷ) = вҲ‘[рқ’Ҹ=0,вҲһ] рқ’ҷвҒҝ
     в—ү Then we integrate term by term.
     в—ү Integrating 1/(1 - рқ’ҷ) produces:
          в—Ӣ -Ln(1 - рқ’ҷ)
     в—ү Integrating the power series produces:
          в—Ӣ рқ’ҷ + рқ’ҷВІ/2 + рқ’ҷВі/3 + рқ’ҷвҒҙ/4 + ...
     в—ү Therefore:
          в—Ӣ Ln(1 - рқ’ҷ) = -вҲ‘[рқ’Ҹ=1,вҲһ] рқ’ҷвҒҝ/рқ’Ҹ

в—Ҹ NМІOМІTМІEМІ 2 вҖ” Meaning of the interval.
     в—ү The interval (-1,1) comes from the geometric series.
     в—ү Since:
          в—Ӣ 1/(1 - рқ’ҷ) = вҲ‘[рқ’Ҹ=0,вҲһ] рқ’ҷвҒҝ
     в—ү only when:
          в—Ӣ |рқ’ҷ| < 1
     в—ү The integrated series representation is valid on the same open interval:
          в—Ӣ -1 < рқ’ҷ < 1
     в—ү Key idea:
          в—Ӣ a power series represents a function only where the series converges.
