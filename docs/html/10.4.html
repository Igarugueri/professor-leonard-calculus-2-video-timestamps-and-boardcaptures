      • but for larger angles such as $\dfrac{3\pi}{8}$ or $\dfrac{\pi}{2}$:

      • reflect it across the polar axis
      • then reflect those two pieces across the line $рқңҪ=\dfrac{\pi}{2}$
    • Final result:
      • the graph is a lemniscate
    • Final interpretation:
      • the curve has the shape of an infinity sign
    • NМІOМІTМІEМІ вҖ” Main idea of this example.
      • unlike the previous rose-curve example, here the square root creates a domain restriction
      • so we cannot keep plotting through all the usual sample angles
      • once the inside of the square root becomes negative, the graph stops there
      • then symmetry is used to generate the rest of the curve

  • [1:37:31]. Tangent lines and slopes for polar curves.
    • Goal:
      • find the slope $\dfrac{рқ’…рқ’ҡ}{рқ’…рқ’ҷ}$ for a curve given in polar form
    • Main idea:
      • polar equations are not exactly parametric equations
      • but they behave like parametric equations with parameter $рқңҪ$
    • Use the rectangular relations:
      • $рқ’ҷ=рқ’“\cos(рқңҪ)$
      • $рқ’ҡ=рқ’“\sin(рқңҪ)$
    • Important:
      • here $рқ’“$ is not a constant
      • $рқ’“$ is a function of $рқңҪ$
      • so we must use the product rule
      • and treat $рқ’“$ implicitly as depending on $рқңҪ$

  • [1:39:23]. Polar curves as parametric curves.
    • Since:
      • $рқ’ҷ=рқ’ҷ(рқңҪ)$
      • $рқ’ҡ=рқ’ҡ(рқңҪ)$
    • Then:
      • $\dfrac{рқ’…рқ’ҡ}{рқ’…рқ’ҷ}=\dfrac{\dfrac{рқ’…рқ’ҡ}{рқ’…рқңҪ}}{\dfrac{рқ’…рқ’ҷ}{рқ’…рқңҪ}}$
    • This is the same idea as in parametric calculus,
      • only now the parameter is $рқңҪ$ instead of $рқ’•$
  • [1:40:03]. Deriving $\dfrac{рқ’…рқ’ҡ}{рқ’…рқңҪ}$.
    • Start with:
      • $рқ’ҡ=рқ’“\sin(рқңҪ)$
    • Differentiate with respect to $рқңҪ$.
    • Use the product rule:
      • $\dfrac{рқ’…рқ’ҡ}{рқ’…рқңҪ}=\dfrac{рқ’…}{рқ’…рқңҪ}[рқ’“]\cdot \sin(рқңҪ)+рқ’“\cdot \dfrac{рқ’…}{рқ’…рқңҪ}[\sin(рқңҪ)]$

  • [1:44:04]. Deriving $\dfrac{рқ’…рқ’ҷ}{рқ’…рқңҪ}$.

    • in $рқ’ҷ=рқ’“\cos(рқңҪ)$ and $рқ’ҡ=рқ’“\sin(рқңҪ)$,
      • both factors depend on $рқңҪ$
    • $\cos(рқңҪ)$ and $\sin(рқңҪ)$ clearly depend on $рқңҪ$
    • and $рқ’“$ also depends on $рқңҪ$ because the polar equation has the form $рқ’“=рқ’Ү(рқңҪ)$

  • NМІOМІTМІEМІ 2 вҖ” Why $\dfrac{рқ’…рқ’“}{рқ’…рқңҪ}$ appears.

    • it changes as $рқңҪ$ changes

      • we get $\dfrac{рқ’…рқ’“}{рқ’…рқңҪ}$

    • вқә Then evaluate at the requested angle
