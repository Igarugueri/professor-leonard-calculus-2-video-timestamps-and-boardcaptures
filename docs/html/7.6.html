  • [2:14]. Classification of the two fundamental cases. вҖ“ [рҹ“·image]
    • Informal idea:
      • An improper integral appears when the usual conditions for a definite integral fail.
      • That typically happens because:
        • the interval is infinite,
        • or the function is unbounded on the interval.
    • Case 1:
      • The interval of integration is infinite.
      • It starts at рқ’Ӯ but extends without bound.
    • Case 2:
      • The interval $[рқ’Ӯ,рқ’ғ]$ is finite,
      • but the function $рқ’Ү(рқ’ҷ)$ is unbounded (has an infinite discontinuity) at an endpoint or at some point inside the interval.

  • [8:12]. рҹ§© Example 1 вҖ” Evaluate the integral $\displaystyle \int_{1}^{\infty}\dfrac{1}{рқ’ҷ^{2}}\,рқ’…рқ’ҷ$. вҖ“ [рҹ“·image]
    • 1. Identify why the integral is improper.
      • The function $\dfrac{1}{рқ’ҷ^{2}}$ is defined for every $рқ’ҷ\ge 1$.
      • There is no infinite discontinuity on $[1,\infty)$.
      • The improper behavior comes from the interval extending to infinity.
      • So this is Case 1: an infinite interval.
      • But if we changed the interval to $[-1,\infty]$, then at $рқ’ҷ=0$ the function would be undefined.
        • It would have an infinite discontinuity there.
        • In that situation, a vertical asymptote appears.
        • So Case 2 would also be present.
    • 2. Replace the infinite upper limit with a variable.
      • Consider $\displaystyle \int_{1}^{рқ’ғ}\dfrac{1}{рқ’ҷ^{2}}\,рқ’…рқ’ҷ$, where $рқ’ғ>1$.
      • We do this because we cannot substitute $\infty$ directly into the integral.
    • 3. Visual interpretation.
      • This represents the area under $\dfrac{1}{рқ’ҷ^{2}}$ from $рқ’ҷ=1$ to $рқ’ҷ=рқ’ғ$.
      • Then we study what happens as $рқ’ғ$ grows without bound.
    • 4. Evaluate the finite integral first.
      • $\displaystyle \int_{1}^{рқ’ғ}\dfrac{1}{рқ’ҷ^{2}}\,рқ’…рқ’ҷ$
      • $=\displaystyle \int_{1}^{рқ’ғ} рқ’ҷ^{-2}\,рқ’…рқ’ҷ$
      • $=\Big[\dfrac{рқ’ҷ^{-1}}{-1}\Big]_{1}^{рқ’ғ}$
      • $=\Big[-\dfrac{1}{рқ’ҷ}\Big]_{1}^{рқ’ғ}$
      • $=-\dfrac{1}{рқ’ғ}-\left(-\dfrac{1}{1}\right)$
      • $=-\dfrac{1}{рқ’ғ}+1$
    • [15:53]. Use a limit to recover the improper integral.
      • The way we handle вҖңplugging in infinityвҖқ is with a limit.
      • $\displaystyle \lim_{рқ’ғ\to\infty}\int_{1}^{рқ’ғ}\dfrac{1}{рқ’ҷ^{2}}\,рқ’…рқ’ҷ$
      • $=\displaystyle \lim_{рқ’ғ\to\infty}\left(-\dfrac{1}{рқ’ғ}+1\right)$
      • $=0+1$
      • $=1$
    • Final evaluation.
      • The integral converges.
      • $\displaystyle \int_{1}^{\infty}\dfrac{1}{рқ’ҷ^{2}}\,рқ’…рқ’ҷ=1$

  • [18:34]. General notation and definitions of convergence. вҖ“ [рҹ“·image]
    • Formal notation:
      • $\displaystyle \int_{рқ’Ӯ}^{\infty} рқ’Ү(рқ’ҷ)\cdot рқ’…рқ’ҷ=\lim_{рқ’ғ\to\infty}\int_{рқ’Ӯ}^{рқ’ғ} рқ’Ү(рқ’ҷ)\cdot рқ’…рқ’ҷ$
      • $\displaystyle \int_{-\infty}^{рқ’ғ} рқ’Ү(рқ’ҷ)\cdot рқ’…рқ’ҷ=\lim_{рқ’Ӯ\to-\infty}\int_{рқ’Ӯ}^{рқ’ғ} рқ’Ү(рқ’ҷ)\cdot рқ’…рқ’ҷ$
    • [21:55]. Definitions:
      • Convergence:
        • the limit exists and is finite
      • Divergence:
        • the limit does not exist, or is infinite

    • Setup:
      • Assume $0\le рқ’Ҳ(рқ’ҷ)\le рқ’Ү(рқ’ҷ)$ on $[рқ’Ӯ,\infty)$.
      • So $рқ’Ү(рқ’ҷ)$ is above $рқ’Ҳ(рқ’ҷ)$ on the interval.
    • Fundamental comparison:
      • If $\displaystyle \int_{рқ’Ӯ}^{\infty} рқ’Ү(рқ’ҷ)\cdot рқ’…рқ’ҷ$ converges,
      • then $\displaystyle \int_{рқ’Ӯ}^{\infty} рқ’Ҳ(рқ’ҷ)\cdot рқ’…рқ’ҷ$ also converges.

        • if the bigger area is finite,
        • then the smaller area must also be finite.
    • Reverse comparison:
      • If $\displaystyle \int_{рқ’Ӯ}^{\infty} рқ’Ҳ(рқ’ҷ)\cdot рқ’…рқ’ҷ$ diverges,
      • then $\displaystyle \int_{рқ’Ӯ}^{\infty} рқ’Ү(рқ’ҷ)\cdot рқ’…рқ’ҷ$ also diverges.
      • Intuition:
        • if the smaller area is already infinite,
        • then the bigger area must also be infinite.
    • Important conditions:
      • The comparison is stated for nonnegative functions.
      • That is why we require $0\le рқ’Ҳ(рқ’ҷ)\le рқ’Ү(рқ’ҷ)$.

      • The implications only work in these directions.
      • You may conclude:
        • bigger converges вҮ’ smaller converges
        • smaller diverges вҮ’ bigger diverges
      • But you may not conclude:
        • smaller converges вҮ’ bigger converges
        • bigger diverges вҮ’ smaller diverges
    • Main idea:
      • Instead of integrating a difficult function directly,
      • compare it with a simpler function whose improper behavior is already known.
